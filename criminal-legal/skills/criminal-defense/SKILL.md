# 刑事辩护全流程工具集

围绕一名辩护人或诉讼代理人从接案到结案的真实工作链条，把常见任务拆成可单独调用的模块。用户提出需求后，先用下方路由表定位最贴近的模块，再按"通用辩护工作流"推进，最后参照对应模块的文书要点出稿。多个需求并存时可串联调用。

## 使用方式

1. 读取用户描述，匹配"能力路由表"中的关键词，确定一个或多个目标模块。
2. 信息不足时先追问关键事实（罪名、办案阶段、强制措施、是否认罪、已有材料），不要凭空假设。
3. 按"通用辩护工作流"的四步推进，最终套用模块文书要点生成成稿。
4. 涉及具体刑法、刑事诉讼法及司法解释条文时，只给出条文指向并提示用户核实现行有效版本，绝不臆造条号、罪名构成或量刑数字。

## 通用辩护工作流

无论调用哪个模块，均沿四个阶段推进，按需取舍：

- 第一步 事实与材料梳理：确认罪名、案情脉络、当事人基本情况、强制措施与羁押天数、办案机关与当前诉讼阶段、已掌握的证据与文书。缺口处明确标注"待补"。
- 第二步 法律与证据分析：从构成要件入手判断罪与非罪、此罪与彼罪；逐项评估证据的合法性、真实性、关联性与证明力；识别程序瑕疵、事实疑点与量刑情节；据此确定辩护方向（无罪/罪轻/程序）。
- 第三步 产出文书：套用对应模块的文书要点成稿，结构清晰、论点与证据对应、留白处给出填写指引。
- 第四步 复核交付：检查逻辑自洽、编号一致、引用与事实匹配；标注须与当事人确认的关键决策；提示需进一步核实的条文与待补材料。

## 各模块文书要点（内联）

下列要点为通用骨架，按个案增删；引用条文一律先核实再落笔。

### 模块1 案件评估与法律意见书
- 框架：委托事项与背景 → 案件基本情况（当事人、罪名、阶段）→ 构成要件逐项分析 → 证据初步评估（现有证据证明力、缺口与疑点）→ 定罪与量刑风险 → 辩护方向建议（无罪/罪轻/程序可行性）→ 下一步工作建议。
- 要害：判断须有依据，不下绝对结论；区分"已知事实"与"推断"。

### 模块2 会见准备
- 框架：会见目的与本次重点 → 待核实事实清单 → 程序权利告知（如实陈述与辩解权、申请回避、申请取保等）→ 拟提问题（分案情、证据、身体状况、诉求）→ 风险提示（如实陈述、避免串供）→ 会见记录留存要点。
- 要害：尊重当事人知情权与选择权，记录客观完整。

### 模块3 阅卷与卷宗梳理
- 框架：卷宗目录与册数清点 → 程序卷/证据卷分类摘要 → 关键证据逐项摘录（来源、形式、证明对象）→ 言词证据要点比对（前后矛盾、与客观证据冲突）→ 程序节点时间线 → 疑点与突破口汇总。
- 要害：忠实摘录卷内内容，区分"卷载"与"评注"。

### 模块4 证据整理与质证
- 框架：按控方证据、辩方证据、待收集证据分类编号（如P/D/T 前缀）→ 逐项标注名称、类型、证明对象、证明力评估 → 三性质证要点（合法性/真实性/关联性）→ 证据间矛盾与疑点汇总 → 待补证据方向与优先级。
- 要害：质证意见落到具体证据编号，避免泛泛而谈。

### 模块5 庭审与发问提纲
- 框架：争议焦点与辩护策略 → 法庭调查（对控方证据的质证逐项、辩方举证顺序与说明、对被告人/证人/鉴定人的发问提纲）→ 法庭辩论（辩护论点、预判公诉观点及应对）→ 量刑意见 → 突发情况应急预案。
- 要害：发问设计围绕待证事实，避免诱导性与无目的提问。

### 模块6 辩护意见与辩护词
- 框架：起诉指控概述 → 辩护总观点 → 分论点逐一展开（事实、证据、法律依据三位一体）→ 量刑从轻减轻情节归纳 → 结论与请求。
- 要害：每一论点须有证据或条文支撑；条文指向待用户核实。

### 模块7 取保候审与强制措施
- 框架：申请主体与对象 → 符合取保/羁押必要性审查的事由（社会危险性、身体状况、案件进展、认罪态度等）→ 担保方式（保证人/保证金）→ 申请理由论证 → 配套承诺事项。
- 要害：理由围绕"无继续羁押必要"展开，结合个案情节。

### 模块8 非法证据排除
- 框架：申请排除的证据范围 → 取证违法的具体情形与线索（刑讯逼供、威胁引诱、非法搜查扣押等）→ 初步证据材料（伤情、同步录音录像缺失、讯问时间地点异常等）→ 申请启动调查程序的理由 → 排除后对案件事实的影响。
- 要害：聚焦取证程序合法性，提供可查的线索而非主观推断。

### 模块9 认罪认罚
- 框架：认罪认罚的自愿性与明智性核查 → 指控事实与罪名确认或异议 → 量刑协商要点（幅度、附条件）→ 具结相关权利义务告知 → 与从宽幅度对应的情节梳理。
- 要害：确保当事人在充分知情下自愿选择，保留对量刑建议的协商空间。

### 模块10 量刑分析与刑期测算
- 框架：法定刑幅度 → 基准刑确定依据 → 量刑情节逐项（从重/从轻/减轻及调节方向）→ 测算过程 → 羁押折抵 → 数罪并罚（如有）→ 缓刑适用条件评估。
- 要害：法定刑幅度、量刑情节调节比例、缓刑条件等均须依据现行有效规定核实，不得编造具体数值。

### 模块11 刑事和解与谅解
- 框架：和解可行性与适用范围评估 → 赔偿与补偿方案 → 谅解书要点（自愿出具、谅解范围、对量刑的意义）→ 和解协议条款 → 履行与回执留存。
- 要害：和解须出于双方真实自愿，避免胁迫，注意可和解案件的范围限制。

### 模块12 上诉、二审与申诉再审
- 框架：一审裁判问题归纳（事实认定、证据采信、法律适用、程序、量刑）→ 上诉/抗诉请求 → 理由分项论证 → 二审注意"上诉不加刑"等程序要点 → 申诉再审则梳理符合再审事由的新证据或法律错误。
- 要害：紧扣裁判文书具体认定与法定上诉/再审事由展开。

### 模块13 被害人代理
- 框架：被害人诉求与权利梳理（控告、申请、知情、求偿）→ 事实与证据支持 → 附带民事赔偿范围与计算 → 对量刑的意见 → 与办案机关沟通要点。
- 要害：在被害人立场上主张权利，赔偿范围依法核实。

### 模块14 涉案财物
- 框架：涉案财物清单（查封、扣押、冻结情况）→ 权属与来源分析（合法财产/违法所得/混同）→ 处置异议或返还申请理由 → 案外人/善意第三人权益主张 → 程序与救济途径。
- 要害：区分应予追缴、退赔与应当返还的财物，主张落到具体物项。

## 通用注意事项

1. 条文核实：所有刑法、刑事诉讼法及司法解释的条号、罪名构成、量刑标准均须核对现行有效版本后引用，绝不臆造。
2. 三性贯穿：证据相关工作始终围绕合法性、真实性、关联性展开。
3. 程序优先意识：重视程序性辩护，非法证据排除与管辖、回避等程序问题常为突破口。
4. 量刑精细化：尽量量化从轻、减轻情节，并核实其法律依据。
5. 当事人确认：辩护策略、认罪认罚、和解、上诉等重大决策须与当事人充分沟通并取得同意。
6. 文书规范：文书结构、称谓、引用须符合办案机关与法院的通行格式要求。
7. 保密义务：严格保守在执业活动中知悉的案件信息与当事人隐私。
