# 量刑测算（五步法）

## 用途

对已认定构成犯罪的个案，按"量刑起点 → 基准刑 → 各情节按比例调节 → 拟宣告刑 → 缓刑判断"的顺序，给出可解释、可复核的刑期测算与量刑建议参考。量刑不是纯数学运算，而是在罪责刑相适应原则统摄下，把法定刑幅度、案件事实与各量刑情节折算为相对合理的刑罚量区间。

## 总则与高风险提示（务必先读）

- 本工作流给出的是**测算参考**，最终刑罚由法院在法定刑幅度内依法裁量；任何结论均不构成对裁判结果的承诺。
- 所有量刑幅度、起点刑取值、调节比例**必须注明所依据的条文或量刑指导意见／实施细则**，并提示用户以**现行有效规定及办案地最新细则**核实；不同地区、不同时期的标准存在差异。
- 凡涉及具体百分比、起点取值，**一律给出区间而非单一确定数**，并说明取值理由。
- **绝不臆造**任何条文、比例或数额标准。规则不明、案情要素缺失时，应明确指出需要用户补充或核实，不得用假定数据掩盖空缺。
- 调节比例不能绝对化，须一案一议；同一情节在不同案件中的从宽／从严幅度可能不同。

## 工作流

严格按下列五步顺序执行，不可跳步。每步先确认所需事实是否齐备，缺失则向用户提问。

### 第一步：确定量刑起点

量刑起点是仅依据该罪名**基本犯罪构成**对应法定刑幅度确定的"裸刑"，此时**不考虑**自首、立功、退赃、认罪认罚等任何具体情节。

要点：
1. **裸刑**：起点只反映基本构成，不掺入具体情节。
2. **取点**：在法定刑幅度内选定一个具体数值（而非区间），以便后续折算；通常落在幅度的**中段偏下**至接近下限处。
3. **依据**：有量刑指导意见或办案地实施细则规定的，按其规定；无明确规定的，依基本构成事实在法定刑幅度内合理取点，并说明理由。
4. 须先向用户确认：罪名、对应法定刑幅度、是否有专门的量刑细则。

### 第二步：在量刑起点上确定基准刑

基准刑 = 量刑起点 + 增加刑罚量。在起点基础上，根据本案**犯罪数额、犯罪次数、危害后果**等影响构成的事实增加相应刑罚量。

- 量刑起点只含基本构成；基准刑则纳入了本案具体的数额／次数／后果。
- 增加刑罚量的换算标准通常由办案地实施细则规定，须查阅**办案地最新细则**并注明出处；无细则时按比例合理估算并说明理由，给出区间。

### 第三步：以各量刑情节按比例调节基准刑

这是核心环节，分两轮进行——先连乘类情节，后加减类情节。

**第一轮：连乘类情节（多为影响罪责的修正情节）**

适用于犯罪形态与责任能力类情节，例如：
- 犯罪形态：预备、未遂、中止；
- 共同犯罪地位：从犯、胁从犯、教唆犯；
- 主体／责任：未成年人、已满七十五周岁的人、限制责任能力人、又聋又哑或盲人；
- 正当化事由过当：防卫过当、避险过当。

公式（逐项相乘）：

```
B = 基准刑 × (1 − a₁%) × (1 − a₂%) × …
```

各 a 取值须注明所依情节的法定从宽幅度区间（如未遂"可以从轻或减轻"、从犯"应当从轻、减轻或免除"等）及对应指导意见，给区间。

**第二轮：加减类情节（一般从宽／从严情节）**

适用于自首、坦白、立功、累犯、当庭认罪、退赃退赔、刑事和解、赔偿谅解、**认罪认罚**等。

公式（同向相加、逆向相减后一次性作用于 B）：

```
C = B × (1 − 从宽合计% + 从严合计%)
  = B × (1 − m₁% − m₂% … + n₁% + n₂% …)
```

- 从宽情节（自首、坦白、立功、退赃、和解、谅解、认罪认罚等）→ 减少比例；
- 从严情节（累犯、累罚、对象特殊等）→ 增加比例；
- **认罪认罚是具有相对独立性的从宽情节，单独计列，不得与坦白等重复或遗漏。**
- 每个比例均须标注依据条文／指导意见，并给区间。

### 第四步：综合校准，得出拟宣告刑

前三步得到的是一个"点"或窄区间，还需作整体校准后形成拟宣告刑：
1. **全案审视**：结合主观恶性、人身危险性、社会危害性整体衡量计算结果是否相称。
2. **类案平衡**：参照同类案件的通常处理结果，避免畸重畸轻。
3. **特殊情形处理**（拟宣告刑须始终受罪责刑相适应原则制约）：

| 情形 | 处理 |
|------|------|
| 计算值趋近于零或为负 | 校准至合理低值；具备条件的提示考虑不起诉／免予刑事处罚 |
| 拟宣告刑低于法定最低刑，且**无**法定减轻情节 | 以法定最低刑为准 |
| 拟宣告刑低于法定最低刑，且**有**法定减轻情节 | 可按拟宣告刑确定 |
| 拟宣告刑高于法定最高刑 | 以法定最高刑为准（除非依法加重） |
| 数罪 | 先分别测算各罪，再依并罚规则合并 |
| 结果明显失衡 | 依罪责刑相适应原则回调并说明 |

若最终要给"幅度型"量刑建议，也应先得出中心点再上下浮动，而非一开始就用宽区间掩盖测算。

### 第五步：缓刑（及非监禁措施）判断

在拟宣告刑确定后，单独判断是否可能适用缓刑或其他非监禁处理：
- **门槛**：核对刑种与刑期是否落入可宣告缓刑的范围（如一般为拘役、三年以下有期徒刑），并注明所依条文及现行标准。
- **实质条件**：综合犯罪情节、悔罪表现、有无再犯危险、对所居住社区有无重大不良影响等作出判断。
- **排除情形**：核对是否属于依法不适用缓刑的对象（如累犯、犯罪集团首要分子等）。
- 结论给出"可能适用／不宜适用／需进一步评估"三档之一，并说明理由与待核实事项；最终是否适用由法院裁量。

## 输出格式

向用户呈现时采用以下结构：

**一、案件基本信息**
- 罪名、法定刑幅度、是否有可适用的量刑细则、办案地。

**二、五步测算过程**

```
第一步 量刑起点：[取值/区间]  依据：[条文/细则；取点理由]
第二步 基准刑：[值/区间] = 起点[值] + 增加刑罚量[值]  依据：[数额/次数/后果换算出处]
第三步 调节基准刑：
  连乘类情节：[逐项列情节+比例区间+依据]
    B = 基准刑 × (1−a₁%) × (1−a₂%) … = [B 区间]
  加减类情节：[逐项列情节+比例区间+依据，认罪认罚单列]
    C = B × (1 − Σ从宽% + Σ从严%) = [C 区间]
第四步 综合校准：[全案/类案判断；特殊情形处理]
       拟宣告刑：[刑种 + 刑期区间]
第五步 缓刑判断：[可能/不宜/待评估] + 理由 + 待核实项
```

**三、风险提示与待核实事项**
- 须用户确认或补充的事实、数额、情节。
- 各比例、起点取值所依规定可能因地区／时间不同而存在差异，提示以现行有效规定核实。
- 本测算为参考，最终量刑由法院依法裁量。

## 示例

> 以下数值仅为演示折算方法，**非真实标准**；实际办案中所有比例与起点均须以现行规定及办案地细则核实。

**示例一：盗窃（仅加减类情节）**
- 案情：盗窃数额较大，有自首，认罪认罚，初犯。
- 第一步 起点：法定三年以下，中段偏下取约 9 个月（依细则核实）。
- 第二步 基准刑：按超出"数额较大"部分换算增加，约 18 个月（依办案地细则区间）。
- 第三步：无连乘类情节，B≈18 个月；自首（约减 20%–30%）+ 认罪认罚（约减 10%–20%），C ≈ 18 ×(1−0.30…−0.15) ≈ 10–12 个月。
- 第四步：参照类案，落在合理区间，拟宣告刑约有期徒刑 10–12 个月。
- 第五步：刑期在缓刑可考虑范围，结合悔罪与再犯危险，提示"可能适用缓刑，待社区影响评估核实"。

**示例二：故意伤害（连乘+加减情节）**
- 案情：致轻伤，系从犯，犯罪未遂，自首，已赔偿取得谅解。
- 第一步 起点：致轻伤、三年以下，取约 12 个月。
- 第二步 基准刑：按伤害后果增加，约 18 个月。
- 第三步：连乘类——从犯（约减 20%–40%）×未遂（约减 20%–40%），B ≈ 18×0.7×0.7 ≈ 8.8 个月（取中值演示）；加减类——自首（约减 20%–30%）+赔偿谅解（约减 20%–30%），C ≈ 8.8×(1−0.4…) ≈ 5–6 个月。
- 第四步：拟宣告刑约 5–6 个月，未低于法定最低刑，无需特殊回调。
- 第五步：短期自由刑，结合谅解与再犯危险，提示"可能适用缓刑，待评估"。

**示例三：测算值趋零的处理**
- 案情：交通肇事致轻伤，自首、赔偿谅解、认罪认罚，从宽情节叠加致计算结果趋近于零。
- 第三步结果趋零；第四步校准时，对具备条件者提示考虑相对不起诉（审查起诉阶段）或免予刑事处罚（审判阶段），并说明判断依据与待核实事项。
