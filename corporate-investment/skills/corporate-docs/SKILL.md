# 公司治理文件起草助手

> 协助起草公司设立与运营中的核心治理文件，确保结构完整、要素齐备、表述规范。

## 重要原则（每次起草前默读）

1. **以现行法为准**：表决比例、法定多数、召集通知期限、登记备案要求等法定事项，均须提示用户以现行《公司法》及配套行政法规、地方登记规则核实后确认，不得凭记忆直接落定。
2. **绝不捏造条文**：引用任何法律条文时，只能写明"依据《公司法》相关规定"或留出条号占位符 `[条号待核]`，不得编造具体条款序号或文字。
3. **硬信息用占位符**：公司名称、注册资本、股东姓名/名称、出资额、出资比例、日期、住所等具体信息一律以方括号占位符（如 `[公司全称]`、`[股东A]`、`[出资额]`、`[YYYY年MM月DD日]`）呈现，由用户填入，不得臆造。
4. **中性免责**：每份交付文件末尾附一句中性提示——本文件为依据通用规则生成的草稿，正式使用前应结合具体情况并经专业人员审核，不构成法律意见。

## 一、文书类型路由

根据用户诉求识别目标文书类型，路由到对应章节：

| 文书类型 | 触发场景 | 适用情形 |
|---------|---------|---------|
| 股东会决议 | 股东会决议、股东决定、股东会 | 公司重大事项由股东（会）作出决定 |
| 董事会决议 | 董事会决议、董事会 | 设董事会的公司就经营管理事项作决定 |
| 公司章程 | 公司章程、章程、章程修正案 | 设立公司或修订章程 |
| 股东协议 | 股东协议、股东之间的协议、投资协议 | 股东就权利义务、治理安排另行约定 |
| 出资协议 | 出资协议、发起人协议、合资协议 | 设立阶段约定各方出资与设立事宜 |
| 增资协议 | 增资协议、增资扩股、引入投资人 | 新增注册资本、引入新股东 |

无法判断时，先向用户澄清拟办事项与公司类型（有限责任公司/股份有限公司），再路由。

## 二、分步工作流

### 第 1 步：信息采集

向用户确认以下要素（缺失项以占位符保留，不臆造）：

- 公司基本情况：全称、类型（有限/股份）、注册资本、住所、是否设董事会/监事会或其替代机构。
- 当事主体：股东/发起人/投资人名单、各自出资额与持股比例、法定代表人。
- 拟决议或拟约定的具体事项：例如选举/更换董事、修改章程、增资、利润分配方案、对外担保、转让股权等。
- 时间与程序信息：会议时间地点、通知方式、表决方式、签署日期。

### 第 2 步：确定文书框架

依据文书类型选取下方对应模板，搭建章节骨架。

### 第 3 步：填充实质内容

- 表决事项与表决比例：区分"普通事项"与"特别事项"（如修改章程、增减注册资本、合并分立解散、变更公司形式等通常需更高比例通过）。**具体比例须提示以现行《公司法》及公司章程为准核实**，模板中以 `[依现行法及章程确定的比例]` 占位。
- 法定程序：召集通知、出席与表决、决议生效条件、记载与存档，均提示按现行法核实期限与要求。
- 治理结构、股权与出资、决策与僵局、利润分配、股权转让与退出等条款，按下方模板要点逐项落实。

### 第 4 步：审核与提示

- 通读全文，检查占位符是否遗漏、各方比例之和是否为 100%、引用是否留作占位。
- 在文末附"起草说明"：列出需用户确认的法定比例/程序、可选条款的取舍建议、以及中性免责语。

### 第 5 步：交付

生成结构清晰的文本草稿；如需 Word 文件，调用文档生成能力输出 `.docx`，文件名建议 `[文书类型]_草稿_[日期].docx`。

## 三、内联模板

### 模板 A：股东会决议

```

股东会决议

时间：[YYYY年MM月DD日]    地点：[会议地点]
召集方式：[现场会议/通讯方式]    通知情况：[已于会议前按章程规定方式通知全体股东]
出席情况：本次会议应到股东 [N] 名，代表表决权 [100%]；实到股东 [N] 名，代表表决权 [比例]。
主持人：[姓名]    记录人：[姓名]

经与会股东审议，依据《公司法》相关规定及《公司章程》，就下列事项作出决议：

一、关于 [议案一名称] 的议案
   决议内容：[具体内容]
   表决结果：同意 [比例]，反对 [比例]，弃权 [比例]；
   本议案[属普通事项/特别事项]，表决比例[达到/未达到]章程及现行法要求的 [依现行法及章程确定的比例]，决议[通过/未通过]。

二、关于 [议案二名称] 的议案
   决议内容：[具体内容]
   表决结果：[同上格式]

（如涉及修改章程、增减注册资本、合并、分立、解散或变更公司形式等特别事项，须单列并适用更高表决比例——具体比例以现行《公司法》及公司章程为准核实。）

本决议自[全体出席股东签字/章程规定的生效条件成就]之日起生效。

全体出席股东（签字/盖章）：
```

### 模板 C：公司章程（要点结构）

> 以下为章程通常包含的章节骨架。各章具体内容须结合公司实际填充，法定必备事项及表决比例以现行《公司法》核实。

```
[公司全称] 章程

第一章 总则
  - 设立依据、公司名称、住所
  - 公司类型、设立目的与经营宗旨

第二章 经营范围
  - 经营范围表述（以登记机关核准为准，留占位）

第三章 注册资本与股权
  - 注册资本：[金额]
  - 各股东出资额、出资方式、出资时间与持股比例（合计 100%）
  - 出资证明与股东名册
  - 出资未实缴/出资不实的责任安排

第四章 股东与股东（会）
  - 股东资格、权利与义务
  - 股东会职权范围
  - 召集与主持、通知期限（以现行法核实）
  - 表决机制：普通事项与特别事项的表决比例 [依现行法及章程确定]
  - 决议方式、回避、记载与存档

第五章 治理机构
  - 是否设董事会及人数、任期、职权、议事规则与表决比例
  - 法定代表人产生与职权
  - 是否设监事会/监事或审计委员会等监督机构及职权

第六章 财务、会计与利润分配
  - 财务会计制度、审计安排
  - 法定公积金提取、利润分配原则与顺序、分配方案的决定程序
  - 亏损弥补

第七章 股权转让与退出
  - 股东内部转让规则
  - 对外转让的同意/优先购买权安排（程序与期限以现行法核实）
  - 股权回购、退出与继承安排

第八章 合并、分立、增减资、解散与清算
  - 各事项的决定程序与表决比例 [依现行法及章程确定]
  - 解散事由与清算组安排

第九章 附则
  - 章程解释、修改程序、生效与备案
  - 未尽事宜的处理（依现行法及补充约定）

全体股东（发起人）签字/盖章：[占位]
制定日期：[YYYY年MM月DD日]

—— 提示：本章程为依据通用规则生成的草稿，法定必备事项、表决比例及登记备案要求请以现行《公司法》及登记机关要求核实，正式使用前应经专业人员审核，不构成法律意见。
```

### 模板 D：股东协议 / 出资协议 / 增资协议

> 三类协议结构高度相通，按实际情形增删条款。出资协议侧重设立阶段出资；增资协议侧重新增资本与投资人引入；股东协议侧重持续期的治理与权利安排。

```
[协议名称]

甲方（[原股东/各方]）：[名称/姓名]，[证件号/统一社会信用代码]，住所：[ ]
乙方（[新投资人，如有]）：[名称/姓名]，[证件号/统一社会信用代码]，住所：[ ]
（如有更多方，依次列明）

鉴于：[简述背景——拟共同设立公司/拟向目标公司增资/各方就治理达成一致等]
各方依据《公司法》相关规定及诚实信用原则，达成如下协议：

第一条 出资与股权
  - 注册资本（或增资后注册资本）：[金额]
  - 各方出资额、出资方式（货币/非货币）、出资时间与持股比例（合计 100%）
  - 非货币出资的评估与权属转移安排
  - 【增资专用】增资价格、是否含溢价及溢价计入资本公积的处理
  - 【增资专用】交割先决条件、交割步骤与时点

第二条 治理结构与决策机制
  - 股东会/董事会的组成、提名权分配
  - 普通事项与重大事项清单及对应表决比例 [依现行法及章程确定]
  - 重大事项一票否决/保护性条款（如约定）
  - 经营管理层的任免

第三条 决策僵局与打破机制（视需要约定）
  - 僵局情形的界定
  - 打破方式：协商、调解、董事/股东轮值、第三方决断、强制收购（如俄罗斯轮盘/德州对决类条款）、退出或解散等
  - 程序与时限

第四条 利润分配
  - 分配原则与顺序、分配的决定程序
  - 是否设优先分配、特殊分配安排（如约定）
  - 不分配情形的处理

第五条 股权转让与退出
  - 内部转让与对外转让规则
  - 优先购买权、共同出售权、领售权（如约定）
  - 回购触发情形与回购价格计算方式
  - 【投资人引入】反稀释、知情权、清算优先权等（如约定，具体设计须合规核实）

第六条 陈述与保证
  - 各方主体资格、出资真实、无权利瑕疵等

第七条 违约责任
  - 出资违约、信息违约等的责任承担方式

第八条 其他
  - 保密、通知、争议解决、适用法律
  - 与公司章程不一致时的衔接安排
  - 生效条件、份数与签署

各方签字/盖章：
甲方：[ ]　　　乙方：[ ]
签署日期：[YYYY年MM月DD日]

—— 提示：本协议为依据通用规则生成的草稿，所涉表决比例、法定程序及特殊条款的合规性请以现行《公司法》及相关规定核实，正式使用前应经专业人员审核，不构成法律意见。
```

## 四、常见易错点检查清单

- 出资比例 / 表决比例之和是否为 100%。
- 特别事项是否与普通事项区分，并提示按现行法核实更高表决比例。
- 召集通知期限、决议生效条件是否留作占位待核，未凭记忆写死。
- 引用法律是否仅写"依据《公司法》相关规定"或留 `[条号待核]`，未捏造条号。
- 章程与股东协议条款是否相互衔接、无冲突。
- 硬信息（名称、金额、日期、住所）是否全部以占位符呈现，未臆造。
- 文末是否附中性免责提示。
