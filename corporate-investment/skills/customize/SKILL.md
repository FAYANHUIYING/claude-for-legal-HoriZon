# /customize

## 何时运行

用户想修改实务画像中的某一项——而不必重新跑整个冷启动访谈，也不必手工编辑文件。

## 要做什么

1. **读取配置。** 读 `~/.claude/plugins/config/claude-for-legal-horizon/corporate-investment/CLAUDE.md`（及上一层 `~/.claude/plugins/config/claude-for-legal-horizon/company-profile.md`）。若插件配置不存在或仍含 `[PLACEHOLDER]`：

   > 你还没有完成设置。请先运行 `/corporate-investment:cold-start-interview`——customize 用于调整一份你已经拥有的画像。

2. **展示可定制清单**，每项附一行当前值摘要：

   - **所务 / 我们是谁** —— 所名、业务、执业设置、管辖地 *（全插件共享——改动经 `company-profile.md` 流转生效）*
   - **领域立场** —— 常代理立场（公司/股东/投资方）、章程决议惯例、意见书格式
   - **风险姿态** —— 保守 / 适中 / 进取
   - **人员与上报** —— 上报联系人、审批阈值、上报链
   - **经验资产** —— playbook / 范本库 / 案库 / 法条 core 的位置与状态
   - **行文与输出** —— 页眉、行文风格、仪表盘偏好
   - **事项工作区** —— 启用状态、当前事项、跨事项开关
   - **集成** —— 检索 / 飞书 / 文档存储状态与回退

3. **询问想改哪一项。**

4. **做出变更。** 展示当前值 → 问新值 → **说明下游会有什么变化**（哪些技能的行为会因此不同）→ 确认后写入配置。

5. **共享画像的变更**（所名、业务、管辖地、执业设置）写入 `~/.claude/plugins/config/claude-for-legal-horizon/company-profile.md` 并注明：此变更影响全部插件。

6. **收尾。** "已完成。你的下一次输出将反映该变更。"

## 护栏

- **绝不删除某个章节。** 用户想"移除"某项时，提议改标为 `[当前不适用——保留以备重新启用]`。
- **标记内部不一致。** 变更使画像自相矛盾时（如风险姿态进取 + 每事必上报），把张力说出来。
- **标记护栏退化。** 飞行前引用检查、来源标签、`[review]` 标记是承重的——不得移除；调整前先说明取舍。
- **一次只改一项。** 不要重跑整个访谈。
