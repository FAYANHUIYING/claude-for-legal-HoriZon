# 商标申请助手

本技能为中国商标申请提供标准化的辅助流程，覆盖"类别规划 → 可注册性初筛 → 申请材料准备"三个环节，产出可直接复用的结构化结论。所有判断均为初步研判，不构成法律意见。

## 适用范围与边界

- 主要面向中国大陆商标申请，引用《中华人民共和国商标法》及其实施条例、商标审查审理规则等国内规范。
- 尼斯分类（国际商标分类）为国际通用分类标准，类别规划部分可参考用于境外申请；但可注册性判断、审查标准等内容仅针对中国大陆语境。
- 涉及马德里国际注册、单一国家申请、抢注、驰名商标认定、跨类大规模冲突等复杂情形时，不在本技能范围内，应转由专业人员深度处理。
- 本技能输出为辅助性初筛意见，不替代执业人员的正式判断，不对注册成功与否作出承诺。

## 触发场景

出现以下需求时启用本技能：

- 咨询"应申请哪些类别""这个商标名能不能注册成功"。
- 需要一份可复用的商标初筛报告，并对风险分级、给出下一步建议。
- 客户已确定商标设计方案，需准备申请材料（商品/服务清单、商标说明）。
- 一般性的商标注册前评估与材料整理。

## 输入收集

下结论前先确认关键信息；信息不足时主动提问，不强行评级。最少需要：

1. 申请主体名称与主营业务（个人/企业、经营范围）。
2. 拟申请商标的形态：纯文字 / 纯图形 / 图文组合，以及具体内容。
3. 已知的目标类别或实际使用场景（产品、服务、销售渠道）。
4. 是否已有设计稿、是否已做过近似查询。

缺失项一律标注"未提及/待补充"，不臆测。

## 工作流程

### 阶段一：类别规划

1. 识别需求类型：`仅类别规划` / `仅可注册性初筛` / `两者组合`。
2. 结合主营业务和使用场景，按尼斯分类确定核心类别与关联防御类别：
   - 核心类别：商品或服务直接所属类别。
   - 关联类别：上下游、配套、宣传推广（如第35类广告销售、第9类电子载体、第42类技术服务）等常见延伸保护类别。
3. 优先覆盖保护范围大的核心商品/服务项，再补充细分项目。
4. 对每个建议类别说明选择理由，便于客户取舍预算。

### 阶段二：可注册性初筛

依据商标法关于禁用、禁注及显著性的一般规则，逐项核查并给出风险判断：

- **绝对理由（禁止注册/使用）**：是否含国家名称、国旗国徽、官方标志、带有欺骗性或不良影响的内容、仅有本商品通用名称/图形/型号、仅直接表示质量等特点、缺乏显著性等。
- **相对理由（在先权利冲突）**：是否与他人在先注册或申请的近似商标在相同/类似商品上冲突，是否可能损害他人在先字号、著作权、外观设计等权利。提示：近似与在先冲突须以官方商标库实际检索为准，本技能不替代检索。
- **显著性强弱**：臆造词 > 任意词 > 暗示性词 > 描述性词，描述性与通用性表达通常显著性弱、驳回风险高。

### 阶段三：申请材料准备

客户确认设计方案后，准备以下材料：

1. **商品/服务清单**：在选定类别下列出规范项目名称，同一类似群归为一组。优先选用尼斯分类规范名称，避免自创表述导致补正或驳回。建议输出为表格，包含字段：序号、类别、类似群、项目名称。
2. **商标说明**：见下节要求撰写。
3. **材料整理**：将申请方案、清单、说明集中归档，文件名建议含商标名、类别、日期，便于检索复用。

## 尼斯分类要点（内联备忘）

尼斯分类将商品/服务分为 45 个大类：第 1–34 类为商品，第 35–45 类为服务。规划时常用关联：

- 第 9 类：科学仪器、计算机软件、可下载电子出版物、APP。
- 第 35 类：广告、商业销售推广、为零售目的的商品展示、特许经营管理（"销售类"防御核心）。
- 第 41 类：教育、培训、文娱、在线出版。
- 第 42 类：技术开发、软件设计、平台技术服务。
- 第 43 类：餐饮、住宿。

**核实提示**：尼斯分类逐年更新版本，类别下的具体可选项目、类似群划分会随版本调整。每次使用须以官方现行版本（含当年增补）为准核实项目名称与类似群编号，**不得凭记忆捏造项目名、类似群号或类别归属**。无法确认时，标注"需核对现行尼斯分类版本"。

## 商标说明撰写

商标说明用于描述商标构成要素及含义，是申请的常规材料，一般控制在 200 字以内。

撰写要点：

1. **声明构成**：说明商标由文字 / 图形 / 文字加图形等何种要素组成。
2. **描述文字**：列明文字内容及字体形式（普通印刷体 / 设计字体 / 艺术字）。
3. **解释含义**：说明文字或整体含义，确无特殊含义的明确声明"无特殊含义"。
4. **图形/组合商标**：先借助视觉识别（直接读取图片）分析其类型、文字内容、图形元素、字体设计特征、配色及整体风格，再据此撰写说明，避免与图样不符。

撰写时如设计细节（寓意、配色意图等）需客户确认，应先提问再定稿，不替客户杜撰含义。

## 输出要求

- 结构顺序：**先结论 → 再依据 → 再风险 → 再下一步行动**。
- 风险分级统一为 **高风险 / 中风险 / 低风险**；信息不足时标注"待补充（信息不足，暂不评级）"。
- 引用法律依据时注明法律名称及条款指向；对所引条文、尼斯分类项目须提示按现行有效版本核实，**绝不捏造条文号、项目名或案例**。
- 缺失信息一律标注"未提及/待补充"，不以推测填充。
- 以 Markdown 结构化呈现，便于直接交付或归档。
- 每次输出必须包含下方免责声明。

## 风险分级标准

| 等级 | 判定情形 | 建议 |
|------|----------|------|
| 高风险 | 触及绝对禁注理由；与在先商标高度近似且类别冲突；显著性明显不足 | 建议调整或更换商标，提交前进一步核查 |
| 中风险 | 显著性偏弱；存在一定近似可能；类别覆盖不完整 | 优化商标设计或清单，必要时补充检索 |
| 低风险 | 显著性较强；未见明显冲突线索；类别规划合理 | 可推进，仍建议正式检索确认 |
| 待补充 | 关键信息缺失，无法评估 | 先补齐主体、商标样式、使用场景等信息 |

风险等级为基于现有信息的初步判断，最终须以官方商标检索及人工复核为准。
