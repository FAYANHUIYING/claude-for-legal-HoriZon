# 民事上诉状与再审申请书撰写

## 概述

本技能帮助使用者在不服原审裁判时，把模糊的"不服"转化为精准的"法律异议"，并据此撰写规范的民事上诉状或再审申请书。核心思路是：先以结构化方式从原审判决中挑出可攻击的错误，再判断走上诉还是再审路径，最后把错误组织成请求与理由，套用内联模板成文。

**免责声明**：本技能输出为文书初稿与思路梳理，供专业核对与决策参考，不构成正式法律意见；所引法律条文、期限与程序要求均须以现行有效法律及司法解释为准，使用前请独立核实。

## 整体工作流

```
读原审判决并挑错 → 判定上诉/再审路径 → 组织请求与理由 → 套用内联模板成文 → 自查与定稿
```

## 第一步：读原审判决，系统挑错

带着"找突破点"的目标意识通读判决，不要被卷宗信息淹没。按以下四个维度逐项排查，把每个疑点记录为"页码/段落 + 原文表述 + 问题类型 + 反驳依据"的条目，便于后续直接转写进文书。

### 维度一：事实认定

- 影响定性的核心事实是否被明确认定，时间、地点、主体、行为、因果等要素是否完整。
- 认定的事实是否有对应证据支撑，是否存在以推测、习惯或常理替代证据的情形。
- 是否遗漏对裁判有决定意义的事实，或对已查明事实作了片面取舍。
- 因果关系、责任比例的认定是否有据。

### 维度二：证据采信

围绕三个关键追问展开：

- 己方提交的关键证据未被采纳，判决是否说明理由，理由是否站得住。
- 对方存在瑕疵的证据被采纳，判决是否给出了合理说明。
- 申请调查取证、鉴定、保全被驳回的，是否有书面或可查的理由。

进一步审查每项证据的真实性、合法性、关联性与证明力，核对质证是否完整，并评估是否存在尚未提交、足以影响结果的材料。

### 维度三：审判程序

- 审判组织是否合法（应回避未回避、参与审理与署名是否一致等）。
- 是否遗漏必须参加诉讼的当事人或第三人。
- 是否剥夺当事人辩论权、质证权等程序权利。
- 送达、公告、开庭等程序是否合规。
- 法院依职权收集的证据是否经过质证。

### 维度四：法律适用

- 所引法律规范、司法解释是否准确、是否现行有效（须逐条核实，不得凭印象引用）。
- 构成要件与本案事实的归入是否准确完整。
- "本院认为"部分的推理链是否环环相扣，前提、过渡结论与最终结论之间有无断裂。
- 判项与诉讼请求是否对应，有无漏判、超判、计算错误或自相矛盾。

### 挑错产出

完成排查后，从所有疑点中筛出一到三个最具突破力的"硬伤"作为主攻方向，其余作为辅助理由。判断硬伤的标准是：错误明确、有据可证、且足以影响裁判结果。

## 第二步：判定救济路径

根据原审裁判所处阶段与错误性质，确定文书类型。

### 选择上诉

- 适用对象：尚未生效的一审判决、裁定。
- 适用情形：认定事实错误、适用法律错误、程序违法、判项不当等。
- 文书类型：**民事上诉状**，由原审法院的上一级法院审理。
- 期限提示：法律对上诉期限有明确规定（判决与裁定期限不同，且自送达次日起算）。**具体天数与起算方式须以现行《中华人民共和国民事诉讼法》为准核实，不要凭记忆填写。**

### 选择再审

- 适用对象：已经发生法律效力的判决、裁定、调解书。
- 适用情形：出现足以推翻原裁判的新证据、原判决基本事实缺乏证据证明、适用法律确有错误、严重违反法定程序等法定事由。
- 文书类型：**再审申请书**，向原审法院的上一级法院或依法有管辖权的法院申请。
- 事由与期限提示：**再审事由必须逐项对照现行《中华人民共和国民事诉讼法》关于申请再审情形的规定核实，严禁捏造或臆造法条与条文序号；申请再审的期限及其特殊起算情形，同样以现行法为准核实。**

### 路径不确定时

若裁判是否已生效、或错误是否构成法定再审事由尚不清楚，先在文书外向使用者说明两条路径的差异与各自门槛，由使用者确认后再成文，避免选错文书贻误期限。

> 提示：本技能仅处理民事文书。若涉及刑事上诉，其上诉主体、期限、审理范围与法律依据与民事程序差异较大，应另行按刑事程序处理，本技能不直接生成刑事文书。

## 第三步：组织请求与理由

### 上诉请求 / 再审请求的写法

- 请求须具体、可执行，能与原判项一一对应（如请求撤销原判第几项、改判为何、或发回重审）。
- 一并主张诉讼费用承担。
- 避免笼统写"请求公正裁判"，应落到明确的处理结果。

### 理由的组织原则

- **攻原审之错，而非复述一审主张**：每条理由都指向第一步挑出的具体错误，引用原判页码或原文，再给出反驳。
- 按"事实—证据—程序—法律"分块或按错误严重程度排序，先打硬伤。
- 每条理由形成"原判如何认定 → 错在何处 → 正确结论应为何 → 依据是什么"的闭环。
- 引用法条时只引现行有效条文，并核对条号与内容一致。

## 第四步：套用内联模板成文

根据第二步选定的路径，使用对应模板。方括号为待填占位，斜杠表示择一。

### 模板 A：民事上诉状

```
民事上诉状

上诉人（原审[原告/被告/第三人]）：[姓名/名称]，[自然人填性别、出生日期、民族、住所；法人或组织填统一社会信用代码、住所、法定代表人姓名及职务]。
[如有委托诉讼代理人，列代理人姓名、单位及联系方式]

被上诉人（原审[原告/被告/第三人]）：[姓名/名称及基本信息，格式同上]

上诉人因[案由]一案，不服[原审法院名称][案号]民事判决，现提起上诉。

上诉请求：
1. 撤销[原审法院名称][案号]民事判决第[ ]项，依法改判[具体请求]；（或：撤销原判，发回重审）
2. 本案一、二审诉讼费用由被上诉人承担。

上诉理由：
一、[第一项错误，如认定事实错误]
[引用原判表述及页码，说明错在何处，给出反驳与正确结论及依据]

二、[第二项错误，如适用法律错误/程序违法]
[同上结构]

[按需续列]

综上，原审判决[认定事实/适用法律/审判程序]存在错误，请求二审法院依法支持上诉人的上诉请求。

此致
[二审法院名称]

上诉人：[签名/盖章]
[年 月 日]

附：本上诉状副本[ ]份；证据材料[ ]份。
```

### 模板 B：再审申请书

```
再审申请书

再审申请人（原审[ ]、[原]审[ ]）：[姓名/名称及基本信息]
[如有委托诉讼代理人，一并列明]

被申请人（原审[ ]、[原]审[ ]）：[姓名/名称及基本信息]

再审申请人因[案由]一案，不服[作出生效裁判的法院名称][案号]已发生法律效力的民事[判决/裁定/调解书]，现申请再审。

再审请求：
1. 撤销[法院名称][案号]民事[判决/裁定]；
2. 依法[改判[具体内容]/发回重审/再审];
3. 本案诉讼费用由被申请人承担。

事实与理由：
本案符合现行《中华人民共和国民事诉讼法》规定的申请再审情形，具体为：

一、[对照现行法核实后填写的具体再审事由，如：有新的证据足以推翻原裁判]
[展开论证：该事由对应的事实、证据及其对裁判结果的颠覆性影响]

二、[第二项再审事由，如：原判决认定基本事实缺乏证据证明 / 适用法律确有错误 / 严重违反法定程序]
[展开论证]

[按需续列]

综上，原[判决/裁定]确有错误，符合法定再审条件，特申请再审，请依法支持申请人的再审请求。

此致
[受理再审申请的法院名称]

再审申请人：[签名/盖章]
[年 月 日]

附：原审法律文书复印件；新证据及相关材料[ ]份；本申请书副本[ ]份。
```

> 模板中所有涉及再审事由表述的部分，须与现行法逐项比对后再定稿；切勿直接照搬模板文字作为法定事由。

## 第五步：自查与定稿

成文后逐项核对：

- 上诉请求 / 再审请求是否具体、可执行，并与原判项对应。
- 每条理由是否都指向具体错误并附依据，而非泛泛重复一审主张。
- 所引法条、案号、当事人信息、金额是否准确一致。
- 再审事由是否已对照现行法核实，无臆造条文。
- 文书类型与救济路径是否匹配，期限是否已提示核实。

定稿前再次提示使用者：上诉期限、再审期限及具体程序要求以现行有效法律为准，须独立核实后再行提交。
