# 案件大事记

## 这个技能做什么

把判决书、裁定书、证据清单等材料里散落的、带时间标记的事实，按时间先后顺序逐一拆出来，整理成一张"事实梳理表"。每一条事实都标明它出自哪份材料、第几页，方便后续核对和引用。

核心原则：**只搬运材料里写明的事实，不替案件下结论、不补充材料里没有的情节**。材料之间相互矛盾的地方，把几种说法并列摆出来，注明各自来源，由办案人员自行判断，技能不替你选边。

## 工作流程

按下面几个步骤推进。每完成一步先把阶段成果发给用户，关键处停下来等用户确认后再继续，不要一口气跑完全部再交付。

### 第一步：通读材料，搭出骨架

1. 把用户提供的全部材料读一遍：判决书/裁定书、证据清单、合同、往来函件、笔录等。
2. 提炼并向用户简要汇报，请其确认理解无误：
   - 当事人及其诉讼地位（原告/上诉人/申请人，被告/被上诉人/被申请人，第三人等）。
   - 案由、基础法律关系。
   - 案件主要事实脉络（一句话概括"大致发生了什么"）。
   - 争议焦点（通常 2 至 4 项）。
   - 是否牵涉多个关联案件，需要分别梳理还是合并到一条时间轴。
3. 这一步的目的是先对齐认知，避免后面方向跑偏。

### 第二步：逐条提取事实节点

1. 以裁判文书"本院查明""经审理查明"等事实认定部分为主线，逐段扫出带时间标记的事实。
2. 再翻一遍证据材料，把文书未展开、但对理解事实有帮助的关键证据节点补进来。
3. 每个节点至少记录四项信息（对应下面的表格字段）：日期、事件、证据来源、重要性。提取时还可在草稿中暂记主体行为、原文摘要、提交方、原件情况等，供填表时取用。
4. **完整性要求**：裁判文书事实认定部分出现的每一个带时间的事实节点，在大事记里都要能找到对应条目，不得遗漏。证据材料中的关键节点也尽量补全。

### 第三步：排序与标注重要性

1. 把全部节点按时间先后排好。日期能精确到日的精确到日；不确定的写到月或年，或写成区间（如"2021.03—2021.08"）并在事件中说明依据。
2. 给每条标注重要性等级（见下方"重要性分级"）。
3. 遇到材料互相打架的情形：不要二选一，把不同来源的说法分行并列列出，各自标清来源，并在重要性或事件中点明"此处存在矛盾"。

### 第四步：成表与交付

1. 按下面的字段把节点填进大事记表，形成可编辑的表格（不要做成图片）。
2. 交付时提示用户重点核对这几处：
   - 裁判文书认定的事实有没有漏掉的节点。
   - 日期排序对不对，有无前后颠倒。
   - 证据来源、页码标注是否准确。
   - 原文引用与概括描述有没有混淆（引用要打引号，概括不打）。
   - 矛盾之处是否如实并列、未被擅自取舍。
3. 根据用户反馈修订，直到事实梳理表准确、完整、可用。

## 大事记表字段

每条事实一行，固定四列：

| 字段 | 填写说明 |
| --- | --- |
| **日期** | 事实发生的时间。精确到日；不确定时写到月/年或写成区间，并在"事件"中注明推断依据。多个节点同日发生时按材料叙述先后排列。 |
| **事件** | 简述谁对谁做了什么。务必写明主体；主体身份首次出现时括注（如"某公司（买受方）"）。引用材料原文用中文引号""标出，自行概括的不打引号。涉及多份材料说法不一致时，分行并列写明各说法。 |
| **证据来源** | 该事实出自哪份材料及具体位置，如"判决书第8页""证据三·付款凭证""往来邮件2021.05.10"。来源不止一处的全部列出；有原件缺失、复印件、需补强等情况一并注明。 |
| **重要性** | 标注该节点对案件的分量，便于快速抓重点（见下方分级）。 |

### 重要性分级

- **关键**：直接影响案件走向或落在核心争议焦点上的事实。
- **重要**：在某一争议焦点项下起支撑作用的证据或情节。
- **关注**：需要特别提醒的信息，如时效节点、矛盾之处、原件存疑、关联案件归属等。
- 其余按时间脉络铺陈的背景性节点可不标，或标"一般"。

可视输出形式（Word 表格、Markdown 表格、电子表格等）按用户需要选择，字段含义不变。

## 几点提醒

- **不臆断**：材料没写明的因果、动机、金额一律不补；拿不准的写"材料未明确"，不自行脑补。
- **如实呈现矛盾**：不同材料对同一事实记载不一致时并列保留，把判断权交给办案人员。
- **来源可回溯**：每条事实都要能顺着"证据来源"列回到原始材料的具体位置，否则不予收入。
- **关联案件**：涉及多案时，在"证据来源"或"事件"中标明该节点属于哪个案件，再统一并入一条时间轴排序，或按用户要求分案制表。

## 适用与边界

- 适用：诉讼准备阶段的案情梳理、上诉/再审的事实核对、为证据交换或庭审整理时间轴、撰写事实陈述前的底稿准备。
- 不适用：合同条款审查、单纯的法律意见咨询、类案检索等非"按时间梳理事实"的需求。

> 本技能产出的事实梳理表是基于所提供材料的整理底稿，供办案参考与核对之用，不构成法律意见，最终事实认定以有权机关的裁判为准。
