# 代理词撰写

## 适用与定位

本技能用于为开庭审理准备代理词（亦称代理意见、辩论意见、法庭辩论提纲）。其核心是站在己方当事人立场，围绕法庭归纳或己方主张的争议焦点，把事实、证据、法律三者编织成一条自洽的论证链，逐点回应并反驳对方观点，最终落到明确的代理意见结论。文书按法庭习惯的代理词结构组织：开头称谓、案由背景、分点展开的代理意见、收尾结论。

边界说明：本技能只生成庭审中宣读或提交的代理词，不生成起诉状、答辩状、上诉状等启动或推进程序的诉状类文书，也不处理合同文本的起草与审查。若需求属于上述范围，应另选合适方法。

## 工作流程

撰写遵循四个阶段，依次推进，前一阶段的产出是后一阶段的输入。

### 第一步：梳理并锁定争议焦点

代理词的全部论证都围绕争议焦点展开，焦点不清则论证必散。

- 先收集案件底数：当事人及诉讼地位（原告/被告/第三人，以及己方代理谁）、案由、诉讼请求与答辩主张、已固定的证据与待证事实、案件所处审级与开庭安排。
- 区分两类焦点：法庭在庭审中已经归纳的争议焦点，应原样采用；法庭尚未归纳的，则由己方依据诉辩主张的对立点自行提炼。
- 把焦点拆成可逐一攻防的小问题，例如"合同是否成立并生效""违约责任如何分担""损失数额如何认定""诉讼时效是否经过"，每个小问题后续单独成段论证。
- 对每个焦点先做一句话的立场预判：己方主张什么结论，主要靠哪一两个支点站住。

### 第二步：组织论证（事实＋证据＋法律三合一）

针对每个争议焦点，按"事实—证据—法律"的内在顺序把论证拼齐，使每一个法律结论都有事实支撑、每一个事实主张都有证据落地。

- **理清事实**：按时间或逻辑顺序把与该焦点相关的关键事实讲清楚，剔除与焦点无关的枝节。事实陈述要中性、克制，避免夹带未经证明的结论。
- **绑定证据**：每一项关键事实主张后面，点明支撑它的证据是哪一份、证明了什么。对己方证据，简述其证明力与指向；对争议较大的事实，从证据的真实性、合法性、关联性三个角度说明为何足以采信。证据务必以卷内已有或当事人确实持有者为准，不得虚构证据名称、内容或出处。
- **嵌入法律**：把事实涵摄到具体的请求权基础或抗辩规范之下，说明构成要件如何被逐一满足或欠缺，从而推出对己方有利的法律结论。引用法律条文、司法解释时须核对现行有效版本与准确条号，宁可留待核实也绝不臆造条文或编号；不确定时，以"相关规定"提示后续补全，并标注需核实。
- 三者要彼此咬合：事实回答"发生了什么"，证据回答"凭什么相信"，法律回答"因此该如何评价"，三句话连起来应是一个完整的推理。

### 第三步：预判并反驳对方主张

代理词不是单方陈述，而是攻防文书，须主动把对方可能的论点摆上台面再逐一击破。

- 站在对方立场，列出其最可能提出的核心主张、所依赖的事实版本与证据，以及援引的法律依据。
- 对每一个对方论点，选择针对性的反驳路径：指出其事实主张缺乏证据支撑或与在卷证据相矛盾；质疑其证据的真实性、合法性、关联性或证明力不足；说明其法律适用错误、援引条文不当或要件并不成立；或退一步指出即便其主张部分成立，也不影响己方结论。
- 把反驳穿插在相应焦点的论证之中，做到"立中有破"，而非另起炉灶单列；对最关键的对立点，可单独成段集中回应。
- 反驳保持就事论事的克制语气，针对主张与证据本身，不作人身攻击。

### 第四步：套用内联代理词模板成稿

将前三步的成果填入下方模板，形成结构完整、可在庭审中宣读或提交的代理词。模板中每个分点对应一个争议焦点，按"己方观点—事实与证据—法律依据—回应对方"的顺序展开；可按案件焦点数量增减分点。

```
代理词

审判长、审判员（人民陪审员）：

____律师接受本案____（原告/被告/第三人）的委托，担任其诉讼代理人参加本案诉讼。庭审前，代理人查阅了案卷材料，参加了庭审调查，现根据本案查明的事实与相关法律规定，发表如下代理意见，供合议庭参考。

一、关于____（争议焦点一，如：合同是否成立并生效）

代理人认为，____（己方就本焦点的明确结论）。

1. 事实方面：____（按时间或逻辑梳理与本焦点相关的关键事实）。

2. 证据方面：上述事实有____（证据名称/编号）予以证明，该证据____（说明证明事项与证明力；必要时从真实性、合法性、关联性角度说明可采性）。

3. 法律方面：依据____（核实后的法律/司法解释条文，未核实处标注待核实），____（说明构成要件如何满足或欠缺，推出有利结论）。

4. 对方主张____，不能成立，理由是：____（针对性反驳：事实无据/证据存疑/法律适用错误/退一步亦不影响结论）。

二、关于____（争议焦点二）

（依上述结构展开：己方观点—事实—证据—法律—回应对方。）

三、关于____（争议焦点三，按需增减）

……

综上，代理人认为，____（综合各焦点，归纳总体代理意见）。据此，恳请合议庭____（如：依法支持/驳回相应诉讼请求，并对责任承担、费用负担等作出有利于己方当事人的认定）。

以上代理意见，请合议庭予以采纳。

代理人：____
____年____月____日
```

## 通用注意事项

- **紧扣焦点**：每一段都应服务于某个争议焦点，与焦点无关的内容一律删去，避免论证被稀释。
- **三性意识**：评价任何一份证据，都从真实性、合法性、关联性三方面着眼，再落到证明力。
- **法条必核**：所有法律条文、司法解释及其条号均须核对现行有效版本后引用；存疑处标注"待核实"，绝不编造法条、案例或证据。
- **事实有据**：任何事实主张都应有证据支撑，不得把推测当作已证事实写入代理词。
- **立场与分寸**：始终站在己方当事人立场组织论证，同时保持就事论事的克制语气，反驳针对主张而非人身。
- **与当事人确认**：涉及诉讼策略与关键事实陈述的重要表述，应与当事人沟通确认后定稿。
- **免责提示**：本技能生成的为供律师审阅修改的代理词草稿，不构成法律意见或对裁判结果的承诺，最终文本须经承办人员审核并对内容负责。
