# 律师函起草

## 概述

本技能帮助起草各类律师函件，包括但不限于律师函、催告函、律师声明、警告函、催款函、敦促函、侵权告知函等。

律师函属于诉讼前或非诉环节的正式沟通文书，其作用在于：以正式、有分量的方式向相对方表明立场，固定事实、阐明法律依据、提出明确主张、设定履行期限，并预告不履行可能产生的法律后果，从而促成相对方自行履行或回到协商轨道，同时为后续可能的诉讼或仲裁保留证据与铺垫。

起草的核心原则是"有理、有据、有节"：
- **有理**——事实陈述客观、准确，不夸大、不歪曲；
- **有据**——主张建立在具体法律依据之上，引用条文须经核实；
- **有节**——措辞庄重克制，施加压力但不越界，避免使用威胁性、侮辱性或不当胁迫的语言，给双方留出回旋与协商的空间。

> 本技能输出为执业辅助性质的文书草稿，不构成法律意见或法律服务承诺，正式发函前应由承办人员结合个案核实事实、证据与法律依据后定稿。

## 工作流程

### 步骤一：明确发函目的与收函主体

在动笔前，先与用户确认以下要素，信息不足时主动追问，不得基于残缺信息臆断：

1. **函件类型与目的**：是催告履行（如催款、催交付）、停止侵权告知、警示违约、澄清声明，还是其他。不同目的决定文书的语气与结构。
2. **发函人**：委托人全称及身份（自然人/法人/其他组织），以及发函人是以委托人名义还是以受托律师名义对外发出。
3. **收函人**：相对方全称、已知的住所或送达地址、联系方式；如有多个相对方，需逐一明确。
4. **送达方式与期望回应**：拟采用的送达方式（邮寄、电子方式、当面送达等）及希望相对方在期限内作出的回应（履行、回函、协商等）。

> 若用户的诉求模糊（如仅说"对方欠钱不还"），应以专业口吻引导其补充事件经过、相关方、争议焦点与期望达到的目的，再继续。

### 步骤二：梳理事实与法律依据

以客观、可被证据支撑的方式组织内容：

1. **事实陈述**：按时间顺序梳理双方之间的法律关系形成、履行经过及争议产生的关键节点。每一项陈述应尽量对应可查证的依据（合同、票据、聊天记录、转账凭证、交付单据等），避免主观评价与情绪化表达。
2. **法律关系定性**：明确双方之间属何种法律关系（如买卖、借款、租赁、服务、侵权等），这是后续主张的落脚点。
3. **法律依据**：找出支撑己方主张的具体规范，包括法律、行政法规及相关规定的条文。

> **法条核实提示**：所有引用的法律名称、条文序号与内容必须逐一核实，确保现行有效、表述准确；不得凭印象引用，更不得虚构或杜撰不存在的条文、案例或事实。无法确认时，宁可作概括表述并提示承办人核实，也不臆造具体条文号。

### 步骤三：提出主张与设定期限

1. **明确主张**：将诉求转化为具体、可执行的要求，避免笼统。例如：
   - 催款类——明确本金金额、计息标准、起算与截止时点；
   - 履行类——明确应履行的具体义务及标准；
   - 侵权类——明确要求停止的具体行为、消除影响或赔偿损失的范围。
2. **设定履行期限**：给出合理、明确的期限（如"自收到本函之日起X日内"），既体现紧迫性，又给相对方留出实际可行的履行时间。
3. **指定回应方式**：写明相对方应如何回应及联系渠道，便于双方衔接。

### 步骤四：陈明法律后果与回旋空间

1. **预告后果**：客观说明相对方逾期不履行可能面临的法律后果（如承担违约责任、赔偿损失、被提起诉讼或仲裁、被申请财产保全、相应费用扩大等），以事实和法律为依据陈述，而非以恐吓口吻威胁。
2. **保留回旋**：在表明立场的同时，表达愿意通过协商妥善解决的态度，为对方主动履行或双方和解留出台阶，避免把话说死。
3. **措辞审查**：通篇复核，确保：
   - 不含侮辱、贬损、恐吓性词句；
   - 不作虚假陈述或无依据的指控；
   - 不以提起刑事控告、举报、曝光等手段进行不当胁迫；
   - 语气正式、克制、有分寸，体现专业性。

### 步骤五：套用内联模板成文

根据函件类型选择下列内联模板组织内容，按需增删段落。生成后向用户呈现，并提示其核对事实与法条后再行定稿、发出。

## 内联文书模板

### 模板一：律师函（通用）

```
律 师 函

致：【收函人全称】
　　【收函人地址】

【发函人全称】（以下简称"我方当事人"）就【事项简述】一事，委托本所律师致函如下：

一、基本事实
　　【按时间顺序客观陈述法律关系的形成、履行及争议经过，逐项对应证据】。

二、我方意见及法律依据
　　我方当事人认为，【对法律关系的定性及对方行为的评价】。根据《【法律名称】》第【X】条之规定，【引述条文要旨，须核实】。

三、我方主张
　　据此，我方当事人正式要求贵方：
　　1.【具体主张一，如支付款项、停止行为、履行义务等】；
　　2.【具体主张二】。
　　请贵方于收到本函之日起【X】日内履行上述事项，并就处理意见函复本所。

四、特别声明
　　若贵方在前述期限内未予履行或回应，我方当事人将依法采取包括但不限于【提起诉讼/申请仲裁等】在内的法律措施，由此产生的一切法律责任及费用由贵方承担。

　　我方当事人亦愿在合法合理的基础上与贵方友好协商，妥善解决争议。盼贵方予以重视并及时回应。

　　此致

【律师事务所名称】
经办律师：【姓名】　执业证号：【号码】
联系电话：【电话】
【年　月　日】
```

### 模板二：催告函 / 催款函

```
催 告 函

致：【收函人全称】

　　【发函人全称】与贵方之间存在【法律关系简述，如买卖合同/借款/服务合同等】关系。根据【合同/借据/约定】，贵方应于【时间】向我方【履行义务/支付款项】。

　　截至本函发出之日，贵方尚有下列义务未履行：
　　1.【欠付本金/未履行事项及金额或标准】；
　　2.【利息/违约金计算方式，须核实约定与法定标准】。

　　现郑重催告：请贵方于收到本函之日起【X】日内，向我方【履行义务/支付上述款项至指定账户：户名、开户行、账号】，并与我方联系确认。

　　逾期未履行的，我方将依法主张相应权利，包括但不限于要求承担违约责任、赔偿损失及实现债权的合理费用，并保留通过诉讼或仲裁途径解决的权利。如贵方确有实际困难，亦请于上述期限内与我方协商，以便妥善处理。

　　特此催告。

【发函人全称】（签章）
联系人及电话：【　　】
【年　月　日】
```

### 模板三：律师声明 / 警告函

```
律 师 声 明

　　本所接受【委托人全称】的委托，现就【事项，如商标/名誉/著作权/不实信息等】相关事宜郑重声明如下：

一、【陈述委托人享有的权利或事实状态，及相对方涉嫌不当行为的客观描述】。

二、上述行为已涉嫌【侵犯权益/违反约定/违反相关规定】。根据《【法律名称】》第【X】条之规定，【引述条文要旨，须核实】。

三、本所受委托人委托，郑重要求相关方：
　　1.立即停止【具体行为】；
　　2.【消除影响/赔偿损失/其他】。

四、相关方应于本声明发出之日起【X】日内停止前述行为并妥善处理。逾期未改正的，委托人将依法追究相应法律责任。

　　本声明旨在依法维护委托人合法权益，望相关方及社会公众知悉。如有误解，欢迎据实沟通澄清。

【律师事务所名称】
经办律师：【姓名】
【年　月　日】
```

## 通用规则

1. **客观中立**：事实陈述以证据为基础，不夸大、不歪曲，不作无根据的指控或承诺。
2. **法条准确**：所有法律依据须经核实，确保现行有效、引用准确；严禁虚构条文、案例或事实。
3. **措辞得体**：正式、庄重、克制，既施加正当压力又留有回旋，杜绝威胁、侮辱、胁迫性表达。
4. **信息完整**：当事人信息、金额、期限、送达地址等关键要素须准确齐备，缺失的主动向用户核实。
5. **风险提示**：成文后提醒用户，本草稿仅供参考，正式发函前应由承办人员核对事实、证据与法律依据并定稿，必要时评估发函的策略影响与时机。
