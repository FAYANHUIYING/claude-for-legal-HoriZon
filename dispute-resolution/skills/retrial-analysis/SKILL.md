# 民事再审可行性分析

面向已经生效的民事判决，评估当事人申请再审是否具备法定事由与现实可行性。核心逻辑是：先客观还原原审在事实、证据、论理、法律适用四个层面的处理，再把发现的每一处问题对照法定再审事由逐项核验，最后形成可向当事人交付的可行性判断。

> 免责声明：本技能输出为辅助性分析草稿，不构成法律意见，也不替代承办人的独立专业判断；是否申请再审、如何主张事由，须由具备资质的执业人员结合案件全貌与现行有效法律最终决定。

---

## 两种工作模式

| 模式 | 所需材料 | 适用场景 | 输出深度 |
|------|---------|---------|---------|
| 快速筛查 | 仅生效判决书 | 初步接案、快速判断是否值得投入 | 精简版报告，标注推断与缺口 |
| 深度分析 | 判决书 + 原审证据 + 庭审笔录 | 准备正式申请、撰写再审申请书 | 完整版报告，含证据对比与策略 |

### 模式自动判定
- 用户只提供判决书 → 进入**快速筛查模式**。说明："已收到生效判决书，当前进入快速筛查模式。本次仅基于判决书文本分析，证据与质证相关结论为反向推断；如需深度分析，请补充原审证据材料与庭审笔录。"
- 用户提供判决书并附证据/庭审笔录 → 进入**深度分析模式**。说明："已收到完整材料，当前进入深度分析模式，将完成事实梳理、证据对比、质证审查与错误识别的全流程分析。"
- 材料不全但用户坚持要结论时，先按快速筛查出具，并明确列出待补材料。

## 工作流

### 第一步　确认模式与材料清点
逐一登记收到的文件类型与数量，判定模式。判决书超过约 20 页时分批读取，确保不遗漏"经审理查明""本院认为""判决如下"等关键段落。

### 第二步　信息提取

**A. 生效判决书（两种模式均需提取）**
- 案件概况：案号、案由、当事人及其原审诉讼地位、审理法院、审级程序、判决/裁定日期、生效日期。
- 事实认定：将"经审理查明"部分按时间顺序整理为事实链，标注页码出处。
- 法院论理："本院认为"部分逐段拆解，概括每段的论证主张与所依赖的前提。
- 判决主文：完整记录判项。
- 双方主张：原告诉讼请求与事实理由、被告答辩意见（取自"诉称""辩称"段落）。

**B. 原审证据材料（仅深度分析模式）**
- 证据清单：逐份列明证据名称、提交方、证明目的。
- 采信情况：标注法院是否采信，未采信的注明法院给出的理由。
- 整理为证据对比表，便于与判决书的认定相互印证。

**C. 庭审笔录（仅深度分析模式）**
- 当庭陈述：双方核心主张的口头表述。
- 质证记录：各方对对方证据真实性、合法性、关联性的意见。
- 辩论焦点：争议要点与辩论交锋的实质内容。
- 程序事项：是否完整记录、有无程序异常（如缺席、回避、送达争议等）。

> 快速筛查模式下，证据与质证信息只能从判决书论理中反向推断。凡推断所得，统一标注"据判决书推断"；凡判决书未涉及、需另行核实的，标注"需补充原审材料核实"。

### 第三步　错误与瑕疵识别
从四个维度查找原审可能存在的问题，描述时使用"疑似""可能""有待核实"等审慎措辞，保持中立，不预设立场。

1. **事实认定层面**：认定事实与在案证据是否对应；有无证据充分却未予认定的事实；有无认定了却缺乏证据支撑的事实；主要事实是否缺乏证据证明。
2. **证据处理层面**：有无未经质证即作为定案依据的证据；有无采信伪造、变造证据的迹象；对双方证据的采信尺度是否一致；当事人申请调查取证被不当驳回的情形。
3. **论理与逻辑层面**：论证有无前后矛盾、跳跃或武断；"本院认为"与事实认定是否一致；判项有无超出或遗漏诉讼请求。
4. **法律适用层面**：所引法条是否准确、是否现行有效；适用的规则与案件事实是否匹配；有无应当适用却未适用的法律；管辖、审判组织、回避等程序是否合法。

将每个发现按严重程度分级：致命（直接对应法定再审事由、足以撼动结论）／重要（影响实体或程序公正）／一般（瑕疵但难以单独支撑再审）。

### 第四步　法定再审事由逐项核验
把第三步发现的每一处问题，对照《民事诉讼法》关于当事人申请再审的法定事由逐项比对。使用下方核验表，逐条记录"是否命中—支撑依据—命中强度"。

> 重要提示：下表为方法性框架，所列事由仅为常见类型概括，**编号、表述与具体内涵必须以现行有效的《民事诉讼法》及相关司法解释为准**。撰写报告前务必核对当前法律文本，按实际条文逐项落实；严禁凭记忆杜撰条文编号、内容或不存在的事由。如对某项是否成立把握不足，标注"待核实"，不得臆断。

#### 再审事由核验表（按现行法核对后填写）

| 事由类别（以现行法条文为准核对） | 是否命中 | 支撑依据（证据/页码/笔录） | 命中强度 |
|---|---|---|---|
| 有足以推翻原判的新证据 | 命中/不命中/待核实 | | 强/中/弱 |
| 认定基本事实缺乏证据证明 | | | |
| 认定事实的主要证据系伪造 | | | |

| 适用法律确有错误 | | | |
| 审判组织的组成不合法或应回避未回避 | | | |
| 无诉讼行为能力人未经代理／应参加诉讼当事人未参加 | | | |

核验时遵循：一处问题可能同时命中多项事由，应分别列出；命中强度依据材料的充分程度评估，材料越扎实强度越高；命中但材料不足的，归入"需补强"。

### 第五步　可行性结论
综合事由命中情况、命中强度、申请时限三项给出判断：
- **建议申请**：存在一项以上强命中事由，且材料可支撑。
- **可尝试但有风险**：仅有中弱命中，或事由成立但举证不足，需先补强。
- **暂不建议**：未发现成立的法定事由，或仅有难以支撑再审的一般瑕疵。

时限提示：根据现行《民事诉讼法》关于申请再审期间的规定核算从判决生效之日起是否仍在法定期间内，并提示新证据等可能影响起算的特殊情形（具体期间与起算规则以现行法为准）。

### 第六步　后续支持
报告交付后，主动询问是否需要：起草再审申请书要点、开展类案检索、或对报告作进一步修改补充。并提醒当事人注意案件信息保密。

## 报告模板

### 快速筛查模式（精简版）

```
=====================================================
            再审可行性分析报告（快速筛查）
=====================================================
说明：本报告仅基于生效判决书生成，未审查原审证据与庭审笔录，
      证据相关结论为反向推断，可能存在遗漏。如需深度分析，
      请补充原审证据材料与庭审笔录。
生成日期：YYYY年MM月DD日

一、案件基本概况
   案号 / 案由 / 当事人（原审地位）/ 审理法院 / 审级 / 生效日期 / 判项摘要

二、原审事实认定梳理（时间线，标注判决书页码）

三、法院论理摘要与初步问题

四、证据情况（据判决书推断）
   | 证据名称 | 提交方 | 推断证明目的 | 法院态度（采信/未采信/不明） |

五、错误与瑕疵
   致命 / 重要 / 一般，分级列出，标注疑似命中的法定事由

六、法定再审事由核验（据现行法核对）
   逐项填写核验表，记录命中与待核实项

七、结论与建议
   命中事由概览 / 时限是否达标 / 建议（建议补充材料深入分析 / 有一定可能 / 暂不建议）

八、待补材料清单
   □ 原审证据材料  □ 庭审笔录  □ 其他
=====================================================
本报告为辅助分析草稿，不构成法律意见。
=====================================================
```

```

              再审可行性分析报告
=====================================================

   | 证据名称 | 提交方 | 证明目的 | 是否采信 | 未采信理由 | 存在问题 |

   致命 / 重要 / 一般，逐条标注对应的法定再审事由

   逐项填写核验表：是否命中 / 支撑依据 / 命中强度

七、再审可行性结论
   事由命中概览 / 时限检查 / 总体建议（建议申请 / 可尝试但有风险 / 暂不建议）/ 理由

八、诉讼策略建议
   主张事由的取舍 / 证据补强方向 / 再审申请书撰写重点

九、可行性评估说明
   基于事由类型、材料充分度、程序与时限因素的综合判断（定性为主，避免绝对化）

十、下一步行动
   优先准备事项 / 主要风险点与应对
=====================================================
本报告为辅助分析草稿，不构成法律意见，最终决策须结合执业判断。
=====================================================
```

## 执行要点
1. **条文以现行法为准**：所有法定事由、申请期间、程序规则均须对照现行有效的《民事诉讼法》及司法解释逐项核实，绝不凭记忆生成条文编号或内容；把握不足处标注"待核实"。
2. **客观中立**：错误识别只描述疑点与依据，不偏袒任何一方，避免绝对化结论。
3. **出处可溯**：事实、论理、证据相关结论尽量标注材料出处（页码、笔录位置）。
4. **大材料分批**：材料篇幅大时分段读取，确保关键段落不漏。
5. **时限敏感**：主动核算并提示再审申请期间，标注可能影响起算的特殊情形。
6. **保密**：提醒当事人妥善保管案件信息。
