# 和解协议起草

## 用途与边界

本技能用于在纠纷已经发生、各方有意通过协商而非裁判方式终结争议时，起草一份权利义务清晰、金额期限明确、争议一次性了结、且具备可执行性的和解协议或调解协议。

适用：诉讼/仲裁前后的庭外和解、调解组织主持下的调解、债权债务的还款和解、合作终止与一揽子结算等。
不适用：尚未产生争议的普通交易合同（应使用合同起草类技能）；起诉状、答辩状、代理意见等诉讼文书。

> 免责说明：本技能生成的文本为供专业人员审核的工作草稿，不构成法律意见；正式签署或提交前应由具备执业资格的人员结合具体案情核对。

---

## 工作流总览

```
梳理争议与各方诉求 → 设计了结方案 → 设计履行与违约保障 → 内联模板成文
```

每一步结束后向用户复述要点并确认，再进入下一步，避免方向偏差。涉及金额、主体、日期等硬信息，一律不臆造，缺失处用占位符标注待补。

---

## 第一步：梳理争议与各方诉求

目标是把"争什么、各方要什么、底数在哪"问清楚。信息不足时自然追问，不机械罗列问题。

需要确认的要点：

- **当事各方**：全称、统一社会信用代码或身份证件信息、住所、联系人。涉及担保人、共同债务人或第三方的，一并列明。
- **争议由来**：基础法律关系（合同、侵权、债务等）、争议焦点、发生经过与关键时间点。
- **争议现状**：是否已进入诉讼或仲裁？案号、管辖机构、所处阶段（立案、审理、执行）？是否有在先生效文书？
- **各方核心诉求**：每一方最想实现什么、最不能接受什么、可让步空间。
- **争议金额范围**：主张金额、对方认可金额、双方预期落点区间。
- **了结意愿与时限**：是否要求一次性彻底了结？对履行时间有无硬性要求（如年底前、某节点前）？
- **履行能力与担保**：付款方资金来源与到位时间，是否需要担保、抵押、第三方连带责任。

确认后简要复述：当事人 / 争议性质 / 拟了结金额区间 / 是否一次性了结 / 是否已涉诉。锁定后如有变更，须重新确认受影响事项。

---

## 第二步：设计了结方案

把谈判落点转化为可写入协议的条款骨架。

- **争议确认条款**：以中性、不含责任认定倾向的措辞概括争议事实，避免任何一方在文本中"认错"而留下不利证据，除非各方明确同意。
- **了结方案主条款**：明确最终解决方式——付款、分期还款、债务减免、资产交付、股权或权益处置、相互停止某行为等。金额、标的、比例处用占位符精确标注（如付款总额【¥____】）。
- **一次性了结与权利放弃**：写明本协议为对争议的"一揽子、终局"解决；各方就本争议项下的全部请求权（含主债权、利息、违约金、损失赔偿等）相互放弃进一步主张的权利；明确放弃范围与例外（如本协议项下义务本身不在放弃之列）。注意放弃条款应表述清楚、避免歧义，必要时提示用户核实是否存在不可处分的事项。
- **诉讼/仲裁的处置**：已涉诉的，约定撤诉、撤回仲裁申请或申请按和解结案的安排，以及诉讼费、仲裁费、保全费的负担。
- **方案档位（可选）**：对核心条款可准备进取、平衡、底线三档措辞，并标注何种情形下从一档让步到下一档，便于谈判中快速调整。

---

## 第三步：设计履行与违约保障

和解协议的价值在于能落地、能强制执行，本步重点在"履行细节"与"违约后果"。

- **款项与履行方式**：付款币种、账户信息（占位符）、一次性或分期。分期的，逐期列明每期金额、到期日、支付条件。
- **履行期限与节点**：每项义务的起算点、完成期限、交付或验收标准。
- **违约责任**：
  - 加速到期：分期付款中任一期逾期，剩余全部款项视为到期。
  - 违约金或逾期利息：按日或按比例计算，注意约定标准应在合理、可被支持的范围内，过高部分有被调减风险——提示用户核实而非写死上限。
  - 恢复原主张：约定一方根本违约时，另一方有权就原争议恢复全额主张（扣除已受领部分），以遏制"打折和解后赖账"。
  - 担保落实：担保人连带责任、抵押/质押登记、违约时的处置路径。
- **保密条款**：约定和解内容、金额、过程的保密范围与例外（如法律要求披露、向法院/仲裁机构提交、专业顾问知悉），违反保密的责任。
- **争议解决**：本协议自身发生争议时的解决方式（诉讼或仲裁）、管辖机构、适用法律。
- **可执行性增强（重要提示）**：
  - 经调解组织或人民调解委员会主持达成的，提示当事人可依法共同申请**司法确认**，赋予协议强制执行力。
  - 在诉讼或仲裁程序中达成的，提示可请求法院制作**调解书**或由仲裁机构出具**调解书/和解裁决**，直接作为执行依据。
  - 庭外纯民间和解未经上述程序的，一方不履行时通常仍需另行起诉，应向用户说明此局限并建议优先选择可赋强制执行力的路径。

---

## 第四步：内联模板成文

依据前三步确认的信息填充下列模板。所有未确认的硬信息保留占位符【____】，并在文末"待补与待核"清单中汇总。涉及法律条文引用时，仅作提示并要求核实，绝不编造条文编号或内容。

```
                          和解协议

甲方（____）：【名称/姓名】
　　统一社会信用代码/证件号码：【____】
　　住所/住址：【____】
　　联系人及电话：【____】

乙方（____）：【名称/姓名】
　　统一社会信用代码/证件号码：【____】
　　住所/住址：【____】
　　联系人及电话：【____】

（如有担保人、第三方，依上列格式续列）

鉴于：
1. 甲乙双方因【基础法律关系，如：____合同／____侵权事宜】产生争议，
   主要情形为【简述争议由来与焦点，措辞保持中性】。
2.〔已涉诉的填写〕该争议现于【法院/仲裁机构】审理，案号【____】，
   现处于【____】阶段。
3. 双方本着平等自愿、互谅互让的原则，就上述争议达成如下和解，
   以一次性、终局方式了结全部争议：

第一条　争议的了结方案
1.1 【最终解决方式，如：甲方向乙方支付和解款项／债务减免／标的交付等】。
1.2 和解款项总额为人民币【¥____】（大写：【____】）。

第二条　履行方式与期限
2.1 〔一次性〕甲方于【____】前一次性支付至乙方账户【____】。
2.2 〔分期〕按下列计划支付：
　　第一期【¥____】，于【____】前支付；
　　第二期【¥____】，于【____】前支付；……
2.3 支付以【实际到账／____】为完成标准。

第三条　争议的一次性了结与权利放弃
3.1 本协议系双方就上述争议达成的一揽子、终局解决方案。
3.2 各方按本协议履行完毕后，就该争议项下的全部请求（包括但不限于
　　本金、利息、违约金、损失赔偿、费用等）相互不再向对方主张任何权利，
　　亦不再就同一争议提起诉讼或仲裁。
3.3 前述放弃不影响任何一方依本协议主张对方履行本协议义务的权利。

第四条　诉讼/仲裁事项的处理〔已涉诉时适用〕
4.1 本协议生效后【____】日内，【撤诉／撤回仲裁申请／申请按和解结案】。
4.2 已产生的诉讼费、仲裁费、保全费由【____】承担。

第五条　违约责任
5.1 任一期款项逾期支付的，剩余未付款项【视为全部到期／____】。
5.2 逾期的，按【____】标准支付违约金/逾期利息（标准须核实，避免被认定过高）。
5.3 一方根本违约的，守约方有权就原争议恢复全额主张（扣除已实际受领部分）。
5.4 〔有担保的〕担保人/抵押安排：【____】。

第六条　保密
6.1 各方对本协议内容、和解金额及协商过程予以保密。
6.2 例外：法律法规要求、向有权机关提交、向各自专业顾问披露的除外。

第七条　争议解决与适用法律
7.1 因本协议引起的争议，由【____法院管辖／提交____仲裁机构仲裁】。
7.2 本协议适用中华人民共和国法律。

第八条　生效与其他
8.1 本协议自各方【签字／盖章／签字并盖章】之日起生效。
8.2 本协议一式【____】份，各方各执【____】份，效力相同。
8.3 〔可选〕双方拟于本协议生效后共同申请司法确认/请求出具调解书，
　　以赋予本协议强制执行力。

（以下无正文，为签署页）

甲方（签字/盖章）：________________
乙方（签字/盖章）：________________
担保人（如有）：________________
日期：____年__月__日
```

成文后向用户输出：

1. **和解协议正文**（已填入确认信息、保留必要占位符）。
2. **关键设计说明**：了结方案要点、权利放弃范围、违约保障机制、所选可执行性路径及理由。
3. **待补与待核清单**：所有占位符项；需用户核实的事项（违约金标准是否合理、放弃范围是否含不可处分事项、是否申请司法确认或制作调解书、引用的任何法律条文）。

---

## 完成标准自检

成文前逐项核对：
1. 争议范围与了结方案是否对应、有无遗漏？
2. 金额、期限、账户等硬信息是否全部明确或已标占位符？
3. 一次性了结与权利放弃条款是否清晰、无歧义？
4. 违约后果是否足以约束履行（加速到期、恢复主张、担保）？
5. 是否已就可执行性给出路径提示（司法确认/调解书/局限说明）？
6. 是否避免了对各方责任的不当认定、避免编造法律条文？
