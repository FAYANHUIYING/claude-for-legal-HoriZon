# 劳动合同定制生成

## 一、定位与原则

按岗位类型从条款方案库中挑选最优组合，一次成稿，产出一份完整劳动合同。覆盖合同期限与试用期、工作内容与地点、工时制度、劳动报酬、社会保险、保密与竞业限制、服务期、解除与终止等核心模块。

工作准则：

1. **少问多判**——只向用户索取最必要的信息（用工形式、岗位名称、岗位层级），其余由专业判断一次性确定，避免逐条追问。
2. **方案先行**——先按岗位定下整套条款方案再落笔，普通岗位用标准条款，管理岗位叠加增强条款。
3. **留空不臆造**——主体名称、个人信息、起止日期、薪资金额、补偿比例等硬信息一律用占位符（下划线或【】）留空，绝不编造具体数字或单位名称。
4. **以法为准**——试用期上限、不定时工时审批、竞业人员范围与补偿、服务期等强制性事项，须以现行《劳动合同法》及用工所在地的地方规定为准；不确定的提示用户核实，绝不想当然。

## 二、工作流程

### 第1步：收集关键信息（一次性提问）

一次性向用户确认以下要点，能从岗位名称预判的先给出建议默认值：

- **用工形式**：固定期限 / 无固定期限 / 以完成一定工作任务为期限。
- **岗位名称**：如"财务经理""销售主管""行政专员"等，用于匹配岗位特色条款。
- **岗位层级**：高级管理人员（总监/副总/总经理等）走增强方案；普通员工走标准方案。
- **试用期与期限偏好**：默认固定期限并按法定上限配置试用期，允许用户调整。
- **保密与竞业**：仅保密 / 保密加竞业限制 / 均不需要（竞业一般只对接触商业秘密的高管及关键岗位适用，须提示核实）。
- **工时制度**：标准工时 / 综合计算工时 / 不定时工时（后两者须经劳动行政部门审批，合同中以"取得批准为前提"表述）。

不必询问主体全称、个人身份信息、具体薪资、合同起止日、违约金数额等，这些一律留空。

### 第2步：展示方案概要并确认（至多一轮）

把上述选择汇总成一份方案概要，请用户确认或微调，例如：

```
合同方案概要
- 用工形式：固定期限
- 期限/试用期：3年 / 试用期 3 个月（金额、起止留空）
- 岗位/层级：财务经理 / 高级管理人员（叠加增强条款）
- 工时制度：标准工时
- 报酬结构：基本工资 + 绩效，金额留空
- 社保公积金：依法参保
- 保密：是（细化财务类商业秘密范围）
- 竞业限制：是（期限、补偿比例留空，须经核实后填写）
- 知识产权归属：归用人单位
- 岗位特色：财务岗职责清单 + 专属交接清单
确认后将直接生成文档；如需调整请说明。
```

### 第3步：组装并生成文档

按"通用条款骨架 + 岗位层级方案 + 岗位特色条款 + 按需附件"的顺序拼装全文。可直接输出为可编辑文本，或用脚本生成 Word（A4 版面、标题居中黑体加粗、正文宋体小四、1.5 倍行距、条款标题加粗）。生成前完成第五节的合规自查。

## 三、通用条款骨架（普通岗位适用）

以下为可直接采用的条款要点，正文用占位符留空硬信息。

**首部**：合同编号、文书标题、甲方（用人单位）、乙方（劳动者）、签订日期，全部留空。开头引述据《劳动法》《劳动合同法》及相关规定，经平等自愿、协商一致订立。

**第一章 合同期限与试用期**：写明合同类型与起止；试用期按法定上限设置（参见第五节），约定试用期内"不符合录用条件"可解除，并列举不符合录用条件的具体情形（如提供虚假入职材料、未按期办理用工手续或签订合同、不能胜任岗位、未通过培训考核或试用考核等）；约定病休、长假期间试用期中止顺延；约定到期续订的协商与意向通知机制。

**第二章 用工前提**：乙方提供的信息真实，依法可在中国境内就业，接受聘用不违反其对第三方的义务。

**第三章 工作内容与地点**：约定岗位与职责（指向《岗位职责确认书》）；约定法定情形下的单方调岗、医疗期满或不胜任时的调岗及薪随岗变；约定调岗后乙方应配合交接报到，有异议应书面沟通而非擅自缺勤；约定主要工作地点及合理出差义务。

**第四章 工作时间与休息休假**：列出标准工时、综合计算工时、不定时工时三种供勾选，后两种注明"以取得劳动行政部门批准为前提"；约定加班需事先审批、未经批准不计加班；保障每周至少休息一日及法定带薪假期。

**第五章 劳动报酬**：约定按月支付及发薪日（遇休息日提前）；薪资数额另以《薪资确认函》载明；约定岗位变动、未完成任务、违纪受罚等情形下的薪资调整规则；约定依法代扣个税及社保公积金个人部分；约定奖金由用人单位视经营情况自主决定、非必然义务；约定停工待岗待遇依法处理。

**第六章 社会保险与福利**：双方依法参加养老、医疗、失业、工伤、生育保险并缴存住房公积金；保障法定休假及病、伤、生育等期间待遇。

**第七章 劳动纪律**：乙方遵守依法制定的规章制度与员工手册，服从管理、保管财物、恪守职业道德；违纪可依规处理直至解除，造成损失应赔偿；未经许可不得在职期间为竞争性主体兼职或自营同类业务；规章制度可经民主程序合理修订并公示；职务成果及相应知识产权归用人单位。

**第八章 保密**：乙方对用人单位及关联方的商业秘密与个人信息负保密义务，未经书面同意不得使用或泄露；可要求另签保密协议；用人单位因合理运营或法律要求可披露乙方相关信息。

**第九章 变更、解除与终止**：约定法律变更或客观情况重大变化时的协商变更；列明协商解除、用人单位即时解除（试用期不符合录用条件、严重违纪、欺诈订立、严重失职或泄密、被追究刑事责任、获准辞职等）、客观情况重大变化解除（提前30日通知或额外支付一个月工资）、劳动者提前30日辞职、经济性裁员等情形；列明法定不得解除的保护情形（职业病观察、工伤致残、医疗期、孕产哺乳期、临近退休的长期职工等）；列明乙方可即时解除的情形（未依法缴纳社保、强迫劳动、未足额支付报酬、规章违法损害权益等）；约定解除或终止时的工作交接、财物归还、离职证明出具。

**第十章 经济补偿与赔偿**：约定乙方违约或过错致损的赔偿范围；约定用人单位出资培训可另签协议约定服务期及违约责任；约定结算时依法抵扣乙方所欠款项及约定的违约金或赔偿金。

**第十一章 争议解决**：先行协商，不成可提请有管辖权的劳动争议仲裁委员会仲裁，依法可就裁决向法院起诉。

**第十二章 其他**：规章制度与本合同同等效力；可经变更协议或重新订立修改合同；用人单位名称变更不影响合同效力；约定送达条款（以登记的地址、邮箱、企业邮箱、短信等为有效送达方式，信息变更应及时书面告知，未告知致退回视为送达）；约定条款可分性、不弃权、附件效力、与法冲突以法为准、一式两份、签章生效。

**签署栏**：甲方盖章及法定代表人或授权代表签字、乙方签字、双方日期，均留空。

## 四、按岗位的方案差异

### A. 高级管理人员增强条款

在通用骨架基础上替换或增设：

- **聘用与汇报**：明确职务及直接汇报对象，承担相称的其他职责，集团制度与本合同冲突以本合同为准，要求始终专业尽责。
- **利益冲突**：投入主要工作时间与精力于本职，未经批准不得从事其他营利活动，不得收受与业务相关的佣金、回扣、礼品。
- **薪酬增强**：约定年薪/月薪结构、定期薪酬复核（复核非必然加薪）、授权从报酬中扣除应付款项；可设住房/用车/通讯补贴及业绩激励（金额、比例留空，凭合法票据）。
- **带薪年假增强**：按司龄约定年假天数（含法定年假），首年按比例计算，年假使用与结转依法处理。
- **强制休假期**：终止通知期内用人单位可要求其暂不履职或不进入经营场所，期间仍为员工、照常计薪并遵守相应义务。
- **知识产权增强**：履职中主要利用单位资源创作的成果及著作权归单位，约定及时披露与未来转让，并就侵犯第三方知识产权作出赔偿担保。
- **竞业限制增强**：界定竞业禁止期（不得超过法定2年上限）与禁止行为（任职竞争对手、挖角员工、争夺客户供应商）；约定在职后按月支付补偿（比例留空，须经核实符合当地标准）；约定违约责任。竞业人员范围、补偿标准与违约金须以现行法律及地方规定为准，提示用户核实。
- **特定事由即时解除**：列举刑事定罪、欺诈或不诚信、故意泄密、重大过失或不当行为、屡次不执行合理指令、故意违反制度等可立即解除的情形。

### B. 以完成一定工作任务为期限

采用简化文本：首部与双方基本信息、明确为以完成一定工作任务为期限并以任务完成为终止点、工作内容与地点、工时制度（特殊工时须经审批）、按月支付报酬、依法参保、劳动保护、依法解除终止与经济补偿、工作交接、争议处理、份数与签署。注意此类合同不得约定试用期。

## 五、按需插入的条款与岗位特色

**可选通用条款**（按岗位特点择用）：送达方式细化、消极怠工/拒不服从认定（管理岗、销售岗）、加班审批留痕（标准工时）、文件须盖章并法定代表人签字方生效、病假手续要求、多经营场所间工作地点弹性约定。

**岗位特色条款**：每份合同都应据岗位特点，在三处嵌入个性化内容——
- 工作职责（第三章岗位条款内，列出该岗位主要职责）；
- 商业秘密范围（第八章保密条款内，细化该岗位接触的秘密类型）；
- 工作交接清单（第九章交接条款内，列出岗位专属交接事项及未交接的后果）。

常见岗位可据其性质自行生成相应内容，例如：财务岗侧重账簿凭证、银行U盾与印鉴、税务与审计资料；销售岗侧重客户名单与交易记录、在谈项目、渠道与应收账款；技术岗侧重源代码、服务器与密钥、在研方案与技术文档；人事岗侧重员工档案与薪酬数据、在办人事手续、考核与争议材料；市场岗侧重调研与推广方案、媒体与渠道资源、费用台账；法务岗侧重案件材料与法律意见、合同模板、知识产权与外部顾问信息。岗位不在常见之列的，按其职责性质合理拟定职责、保密范围与交接清单。

## 六、生成前合规自查

落稿前逐项核对，发现不合规即提示用户：

| 核查项 | 要求 |
|--------|------|
| 试用期上限 | 合同期3个月以上不满1年→≤1个月；1年以上不满3年→≤2个月；3年以上或无固定期限→≤6个月 |
| 试用期工资 | 不低于约定工资的80%且不低于当地最低工资 |
| 竞业限制期限 | 不超过2年，且仅适用于负有保密义务的高管、高级技术人员及其他人员 |
| 竞业补偿 | 在职后按月支付，标准须达到当地规定下限 |
| 违约金情形 | 仅限服务期与竞业限制两种情形可约定由劳动者承担违约金 |
| 社保约定 | 不得约定放弃或不缴社保 |
| 任务期限/非全日制 | 不得约定试用期 |
| 特殊工时 | 综合计算与不定时工时须经劳动行政部门审批，合同注明以批准为前提 |

具体数值与地方差异请以现行《劳动合同法》及用工所在地规定为准，不确定时提示用户核实，切勿臆造。

## 七、配套附件（按需附上）

- 薪资确认函：单独签署，调薪时无需改动主合同；载明岗位、入职日、试用期、各项税前金额（留空）及代扣代缴、奖金自主、调薪以最新确认函为准等说明。
- 劳动合同变更协议：履行中变更使用。
- 续签协议：到期续订使用。
- 岗位职责确认书：明确职责与考核标准。
