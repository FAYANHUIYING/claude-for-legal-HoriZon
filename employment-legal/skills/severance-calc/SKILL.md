# 劳动经济补偿与赔偿金测算

面向中国大陆一般劳动用工语境，根据工作年限、工资水平与离职情形，分步测算劳动者解除/终止劳动关系时可主张的各项款项。本技能为**单文件、零配置**工具：所有公式与口径均内联于下文，无需外部知识库或任何接口。

输出定位：**测算参考**，用于估算区间与厘清计算逻辑，**非正式法律意见**；各地裁判口径、基数认定与封顶标准存在差异，最终金额以仲裁裁决或法院判决为准。

---

## 使用前先采集的要素

开始计算前，向用户确认以下信息；缺项时明确标注"待补"，并对结果给出区间或留空，不得臆造：

1. **入职日期 / 离职日期** —— 用于计算工作年限（精确到月）。
2. **离职情形** —— 决定适用哪一项（见下文"情形 → 项目对照"）。
3. **月工资构成与金额** —— 计算基数的核心，需区分"应得工资"口径（详见下文基数说明）。
4. **离职前 12 个月的工资明细** —— 有奖金、提成、津贴波动时需逐月列出以求平均。
5. **用工所在地** —— 决定社平工资 3 倍封顶数额与最低工资等地方标准。
6. **是否签订书面劳动合同、起止时间** —— 关系二倍工资。
7. **加班事实与举证情况** —— 加班费需有考勤/审批等证据支撑。
8. **应休未休年假天数** —— 年假折算项。

---

## 情形 → 项目对照（先定性，再套公式）

> 本对照仅用于路由到正确的计算项；是否真正满足某一情形属定性判断，超出本技能范围，应提示用户结合事实与专业意见确认。

| 离职情形 | 可测算项目 |
|---|---|
| 协商一致由用人单位提出解除 | 经济补偿金 N |
| 劳动合同期满终止（单位不续签或降低条件续签被拒） | 经济补偿金 N |
| 用人单位存在欠薪/未缴社保等，劳动者被迫解除 | 经济补偿金 N |
| 医疗期满不能工作 / 不能胜任 / 客观情况重大变化 | N + 代通知金（即 N+1） |
| 经济性裁员 | 经济补偿金 N |
| 用人单位违法解除/终止且劳动者不要求继续履行 | 赔偿金 2N（与 N 二选一，不并得） |
| 用工满一个月未签书面劳动合同 | 二倍工资差额 |
| 存在法定标准外加班 | 加班费 |
| 应休年假未休 | 未休年假工资折算 |

**重要互斥关系**：违法解除赔偿金 2N 与经济补偿金 N **不并列计算**，主张 2N 时不再另算 N。代通知金（"+1"）仅在特定法定情形适用，不可随意叠加到任意解除上。

## 一、工作年限 N 的认定（多数项目的公共变量）

设入职至离职的连续工作时间为基准：

- 满 1 年记 1；
- **超过 6 个月不满 1 年的，按 1 年计**；
- **不满 6 个月的，按 0.5 计**。

示例：3 年 7 个月 → N = 4；3 年 5 个月 → N = 3.5；3 年整 → N = 3。

**年限封顶提示**：对"月工资高于用工地上年度职工月平均工资三倍"的劳动者，其经济补偿的**计算年限最高不超过 12 年**；月工资未达 3 倍线的劳动者，年限一般不设此 12 年上限。是否触发封顶取决于基数与当地社平 3 倍数额的比较，须逐案判断。涉及跨制度年限分段的（如分段计算历史工龄）情况复杂，应标注"[需核实分段口径]"。

## 二、月工资计算基数（高风险口径，必须重点说明）

> **这是最易出错、争议最大的环节。基数口径直接决定全部金额，务必向用户解释清楚并提示核实。**

- **基本口径**：劳动者在劳动合同解除或终止前**12 个月的平均应得工资**。
- "应得工资"通常指**税前、代扣代缴个税及社保公积金个人部分之前**的工资总额，一般包含计时/计件工资、奖金、津贴、补贴等货币性收入；是否包含某项（如年终奖、报销性补贴、加班费本身）各地认定不一，标注"[需核实当地现行标准]"。
- **不满 12 个月**的，按实际工作月数的平均工资计算。
- **上限（社平 3 倍封顶）**：月工资高于**用工所在地上年度职工月平均工资三倍**的，基数按该"3 倍数额"封顶计算。
  - 该"社平工资 3 倍"的**具体数额按地区、按年度发布**，本技能不内置任何数字；使用时一律填 **`[需核实当地现行标准]`**，并提示用户以用工地人社部门/统计部门最新公布为准。
- **下限**：基数原则上不低于当地最低工资标准；具体数额同样标注 `[需核实当地现行标准]`。

计算时请显式写出：
```
月工资基数 = min( 离职前12个月平均应得工资 , 当地社平工资3倍[需核实当地现行标准] )
（若低于当地最低工资标准[需核实当地现行标准]，则按最低工资）
```

## 三、逐项公式与算法

### 1. 经济补偿金 N

```
经济补偿金 = 月工资基数 × N
```
- N 按"第一节"规则取值（含半年/不满半年规则）；
- 月工资基数按"第二节"口径（含 3 倍封顶）；
- 触发 3 倍封顶时，N 同时受 12 年上限约束：`实际计算年限 = min(N, 12)`。

**分步**：① 算工作年限 → N；② 算 12 个月平均应得工资；③ 与当地社平 3 倍比较定基数、判断是否封顶并据此决定 N 是否封 12 年；④ 相乘。

### 2. 违法解除/终止赔偿金 2N

```
赔偿金 = 经济补偿金标准 × 2 = 月工资基数 × N × 2
```
- 适用于用人单位违法解除/终止且劳动者**不要求继续履行**的情形；
- **与 N 互斥**：算 2N 即不再另算 N。

### 3. 代通知金（N+1 中的"+1"）

```
代通知金 = 上一个月的工资 × 1
```
- 仅在法定"提前30日书面通知"情形下、用人单位未提前通知而选择以钱代通知时适用（如医疗期满不能工作、不能胜任工作、客观情况重大变化三类）；
- 注意"+1"的基数通常为**上月工资**，口径可能与经济补偿金的"12 个月平均基数"**不同**，需分别计算，标注 `[需核实当地现行口径]`；
- N+1 总额 = 经济补偿金 N + 代通知金。

### 4. 未签书面劳动合同二倍工资差额

```
二倍工资差额 = 应支付二倍工资期间的每月工资 × 1（即额外一倍）× 计算月数
```
- 适用：用工之日起满 1 个月不满 1 年未订立书面劳动合同；
- **起算**：自用工满 1 个月的次日起；**最长不超过 11 个月**（即除正常发放的 1 倍外，额外再付不超过 11 个月的 1 倍）；
- "额外一倍"的**月基数**按当月应得工资认定，口径争议处标注 `[需核实当地现行标准]`；
- 注意**仲裁时效**对可主张期间的影响，须提示用户核实，标注 `[需核实时效起算]`。

### 5. 加班费

设"加班费计算基数"为劳动者正常工作时间月工资对应的小时/日工资（基数认定本身常有争议，标注 `[需核实当地现行标准]`）：

```
日工资  = 月计薪基数 ÷ 21.75
小时工资 = 日工资 ÷ 8

工作日延时加班费 = 小时工资 × 加班小时数 × 150%
休息日加班费     = 日工资 × 加班天数 × 200%   （已安排同等补休的可不另付）
法定节假日加班费 = 日工资 × 加班天数 × 300%
```
- 21.75 为制度月计薪天数的通用折算值；
- 加班事实与时长需有考勤、审批等证据支撑，举证不足时只列公式、不臆断金额。

### 6. 未休年假工资折算

```
应休未休年假工资 = 日工资 × 应休未休天数 × 300%
其中日工资 = 月工资基数 ÷ 21.75
```
- 该 300% 通常被理解为**包含**用人单位正常支付的当天 100% 工资在内，即"额外再补 200%"，但表述口径各地不一，应标注 `[需核实当地现行口径]`；
- 应休天数与工龄挂钩，须确认劳动者累计工龄对应的年假基准天数。

## 五、风险与合规红线（必须遵守）

1. **绝不捏造数额**：社平工资、社平 3 倍、最低工资、各地分年度标准一律输出 `[需核实当地现行标准]`，并提示用户核实，不得凭记忆填具体数字。
2. **基数口径先讲清**：每次计算前显式说明"月工资基数"的含义、12 个月平均与 3 倍封顶规则，再代入。
3. **封顶双重检查**：触发社平 3 倍封顶时，同时检查 12 年年限上限。
4. **互斥不并得**：N 与 2N 二选一；"+1"仅限法定情形。
5. **低确定性给区间**：基数有争议、口径分地区、证据不足时，给区间或留"待补"，不输出唯一确定值。
6. **数字注明依据**：凡引用的标准、倍数、上限，均说明来自上文哪条规则或注明"[需核实]"。
7. **不越界做定性**：是否违法解除、是否工伤、竞业是否有效等不在本技能范围，应明确告知并建议另行专业判断。
8. **时效提醒**：涉及二倍工资、加班费等可主张期间时，提示劳动仲裁时效影响，标注 `[需核实时效起算]`。
