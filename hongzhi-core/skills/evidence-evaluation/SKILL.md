# 证据三性与证明力评估

## 适用说明

本技能用于在中国法语境下，对民事、刑事、行政诉讼及仲裁中的证据进行体系化审查，输出可供执业者参考的证据评估意见。核心法源包括《民事诉讼法》《刑事诉讼法》《行政诉讼法》及相应司法解释中的证据规定，以及关于民事证据、办理刑事案件排除非法证据、行政诉讼证据等专门规定。

> 中性免责：本技能产出为辅助性分析，不构成正式法律意见。证据能否采信、待证事实能否认定，最终由有管辖权的裁判机关依法决定。AI 无法直接审验原件原物，亦无法替代庭审质证与鉴定程序，分析结论应交由具备执业资格的法律专业人员复核后使用。

## 核心框架

证据审查分两层递进：先判断"证据能力"（能否进入认定，即可采性），再衡量"证明力"（对待证事实的证明分量）。次序不可颠倒——不可采的证据无需也不应评估其证明力。

```
证据能力（可采性）
  ├ 真实性：载体是否真实未篡改 + 内容是否与客观事实相符
  ├ 合法性：取证主体、形式、程序、内容是否均符合法律要求
  └ 关联性：与待证事实是否存在逻辑联系（直接/间接）及其强弱
证明力（证明分量）
  ├ 单证证明力：依证据种类适用差异化评估规则
  ├ 印证关系：多证之间相互印证、补充或冲突
  └ 证据链：全案证据能否就关键环节形成闭合链条
```

## 三大要素与举证责任速览

| 诉讼类型 | 主要证明标准 | 举证责任基本规则 |
|---|---|---|
| 民事 | 高度盖然性（待证事实存在的可能性明显大于不存在；欺诈、胁迫、恶意串通、口头遗赠等适用更高标准） | 谁主张谁举证；法定情形举证责任倒置（如环境污染、产品责任、部分医疗损害）；自认免除对方举证 |
| 刑事 | 排除合理怀疑（证据确实、充分，形成完整证明体系，结论具有唯一性） | 控方对犯罪事实负举证责任；个别罪名存在举证责任转移 |
| 行政 | 行政行为合法性证据应达明显优势 | 被告对行政行为合法性负举证责任，且原则上不得诉中向原告、证人取证；原告对起诉条件、赔偿损失等负责 |

## 分步工作流

### 第一步：搭建评估框架（先于一切实质审查）
- 确认诉讼类型，据此锁定适用的证据规则体系
- 列明全部待证事实，并将每项事实映射到对应证据
- 为每项待证事实标注举证责任归属与适用的证明标准
- 明确案件争议焦点，区分无争议事实与核心争点

### 第二步：采集与分类证据
对每份证据提取：编号/名称、证据种类、来源（谁提供、如何取得）、载体形式（原件/复印件/复制品）、提交时间、拟证明的事实。再按种类、待证事实、提交方、性质（直接/间接、原始/传来、本证/反证）多维归类。

### 第三步：逐项三性审查
对每份证据依次完成三性核验（核验清单见下文），任一要素存在硬伤即可能影响可采性。审查顺序固定为：真实性、合法性、关联性。

### 第四步：单证证明力评级
通过可采性门槛后，按证据种类适用差异化规则评定证明力（高/中/低）。常见提升与减损因素：

- **书证**：原件、由公权机关或公证机构制作、业务中正常形成、记载不利于制作方的内容、签章日期齐全 → 证明力高；复印件无法核对、诉讼中专门制作、仅记载有利于己方的内容 → 证明力低。
- **证人证言**：亲身感知、与当事人无利害关系、出庭接受质证、陈述前后一致且细节具体 → 证明力高；传闻或推测、存在利害关系、仅提交书面证言未出庭、前后矛盾 → 证明力低。
- **电子数据**：经公证保全或可信存证、数据链完整（生成—存储—传输—提取）、具备防篡改校验、可关联到特定主体身份 → 证明力高；仅为截图打印件、片段缺乏上下文、无法证明未被改动或使用者身份不明 → 证明力低。
- **鉴定意见**：机构与鉴定人资质适格、检材来源明确且保管链完整、采用公认方法、过程可复核、结论明确 → 证明力高；资质或检材存疑、方法不规范、结论附条件过多 → 证明力低。
- **当事人陈述**：构成对己不利的自认、前后一致、有他证印证 → 证明力高；孤证无印证、笼统回避、前后矛盾 → 证明力低。

### 第五步：印证关系与证据链分析
- 梳理证据之间是完全印证、部分印证还是相互冲突
- 对冲突证据分析成因（记忆偏差、立场倾向、伪造可能），判断是否触及核心事实，并依证明力规则确定取舍
- 评估证据链是否覆盖每项待证事实的关键环节、有无断裂、间接证据能否形成闭合且排他的逻辑链条

### 第六步：综合判断与结论
针对每项待证事实汇总正反证据，对照适用证明标准作出认定，并归入下列结论之一：证据充分（可认定）、基本充分（细节待补）、证据不足（难以认定）、证据势均力敌（需进一步调查）。对不确定项须明确标注并说明理由。

### 第七步：补强与质证建议
针对薄弱环节给出可操作建议：需补充的证据类型与内容、现有证据的固定保全方式、质证策略（申请证人出庭、申请鉴定或重新鉴定、申请调取等）、举证期限与程序提醒。

## 三性核验清单（逐项勾选）

**真实性**
- [ ] 书证是否为原件？复印件能否与原件核对一致？
- [ ] 物证是否为原物，有无损坏、变质或污染？
- [ ] 电子数据是否具备完整的生成、存储、传输记录，是否经公证或可信存证？
- [ ] 证人证言是否源于亲身感知，有无传闻成分？
- [ ] 鉴定意见的检材是否真实、完整、保管链未断？
- [ ] 视听资料是否为原始载体，有无剪辑、拼接痕迹？
- [ ] 证据是否存在内在矛盾，或与其他已知事实、案件时间线冲突？
- [ ] 是否存在足以推翻其真实性的相反证据？

**合法性**
- [ ] 取证主体是否适格（民事当事人/代理人、刑事侦查人员是否需回避、行政机关是否在作出行为时收集）？
- [ ] 取证程序是否合法，有无刑讯逼供、暴力、威胁、非法搜查扣押、非法窃听偷拍等情形？
- [ ] 证据形式是否合法（书证形式、鉴定机构与鉴定人资质、证人作证能力、笔录见证人签名等）？
- [ ] 是否属于应当排除的非法证据？刑事中以非法方法取得的供述、以暴力威胁取得的证言被害人陈述应绝对排除；实物证据取证违法、可能严重影响司法公正且不能补正或合理解释的裁量排除；民事中以严重侵害他人合法权益、违反法律禁止性规定或严重违背公序良俗方式获取的应排除。
- [ ] 程序瑕疵能否补正或作出合理解释（区分可补正的瑕疵证据与应排除的非法证据）？

**关联性**
- [ ] 证据与待证事实之间是否存在逻辑联系？
- [ ] 关联属直接还是间接？间接证据的推理链是否完整、合理？
- [ ] 关联强弱如何，是否与争议焦点相关？
- [ ] 是否因关联过弱而不具备证据价值？

## 不能单独定案的证据（需他证补强）
- 无法与原件、原物核对的复印件、复制品
- 与一方当事人有利害关系的证人所作的相应证言
- 存有疑点的视听资料、电子数据
- 无正当理由未出庭作证的证人书面证言
- 认知或表达能力受限者（如年幼、相关障碍）所作的与其状况不相当的陈述
- 刑事中缺乏其他证据印证的被告人供述（坚持孤证不能定案）

## 证明力一般优先顺序（民事参考）
1. 公权机关依职权制作的公文书证 > 其他书证
2. 物证、档案、鉴定意见、勘验笔录、经公证或登记的书证 > 普通书证、视听资料、证人证言
3. 原始证据 > 传来证据；直接证据 > 间接证据
4. 出庭质证的证言 > 未出庭的书面证言；无利害关系证人 > 有利害关系证人
5. 多个种类不同而内容一致的证据 > 孤立的单一证据

## 特殊证据要点
- **电子数据**：从生成、存储、传输、提取、展示五个环节审查完整性与可靠性；聊天记录关注账号实名与对话完整性、电子邮件关注邮件头与服务器记录、网页关注是否公证保全、电子签名关注是否符合法定要求、监控录像关注设备状态与时间校准。
- **鉴定意见**：勿因其"专业性"而放松审查，应全面核查机构与鉴定人资质、检材来源与保管链、方法与结论；多份鉴定冲突时可申请重新或补充鉴定、组织鉴定人出庭。
- **域外证据**：境外形成的证据原则上需办理公证认证或履行相关条约规定的证明手续；港澳台地区证据适用特殊确认程序；外文证据须附经有资质机构翻译的中文译本。
- **公证文书**：证明力优先，但有相反证据足以推翻的除外；注意公证程序合法性。

## 常见易错点（务必自查）
- 混用诉讼类型的证据规则（如以民事标准评判刑事证据）——评估前必先确认诉讼类型
- 错配证明标准——结论须明确对照所适用的标准作出
- 遗漏非法证据排除审查——每份证据都应过合法性这一关
- 忽视举证责任归属，对未尽举证责任的一方作出有利认定
- 将证据能力与证明力混为一谈，先后次序颠倒
- 孤证定案，未检查关键事实是否需要补强
- 忽视证据的时间维度，采信事后伪造或与案件无关的材料

## 输出结构建议
1. **评估框架**：诉讼类型、案由、证明标准、举证责任分配、待证事实清单
2. **逐项证据评估**：每份证据的三性结论、证明力等级、是否需补强（可用表格 + 置信度星级 ★☆ 表示）
3. **证据链分析**：印证/冲突关系、各待证事实的证据链状态及是否达标
4. **综合结论**：各待证事实的认定意见与置信度，不确定项的说明
5. **补强与风险**：按优先级排列的补强建议及主要证据风险提示

置信度可用五级星标（★★★★★ 至 ★☆☆☆☆）标注单证可靠性，并对综合结论区分"确定/较确定/一般/不确定"，提示哪些判断依赖裁判者裁量或有待新证据。
