# 法律术语规范化

## 一、定位与免责

本技能帮助把一段文字中的用语调整为准确、统一、合乎法律文体的表达，并对常见的易混术语作辨析。处理对象是含有（或本应含有）法律术语的文本片段，产出为规范化后的文本加一份逐条修正说明。

> 本输出为结构化参考与工作草稿，供专业人员复核，不构成正式法律意见。术语的最终采用应以现行有效的法律、法规、司法解释及官方文本为准；涉及重大权益的文书，应经执业人员核定。

## 二、规范化追求的三件事

- **准确**：所选用语的含义与其指向的法律概念严丝合缝，外延不宽不窄。
- **统一**：同一文本内，同一概念自始至终只用同一个词，不为修辞而换词。
- **得体**：符合法律文书庄重、克制、精确的文风，剔除口语、修辞和情绪化措辞。

## 三、择词的依据顺位

当一个概念有多种可能表达时，按下列顺序确定规范用语：

1. 现行法律、行政法规的原文用语（最高权威）；
2. 最高人民法院、最高人民检察院的司法解释用语；
3. 国家标准或官方规范性文件中的术语；
4. 通行教科书、权威工具书中的通说表达；
5. 裁判文书长期稳定采用的实务惯例。

四项基本守则：法定优先（法律有明文则照用）；前后一致（一概念一术语）；语境贴合（同词在不同法律部门含义可能不同，按语境取舍）；与时俱进（随法律修订更新，避免沿用已废止的旧词）。

## 四、内联易混术语对照表

下表为处理时的核对清单，按法律部门归类。"❌"列为常见的错误或不精确写法，"✅"列为规范写法。

### 民事

| ❌ 易错 | ✅ 规范 | 辨析要点 |
|---|---|---|
| 订金当定金用 | 定金 / 订金 | 定金适用定金罚则、有担保效力；订金仅为预付款，无双倍返还效力 |
| 违约金与赔偿金混用 | 违约金 / 损害赔偿金 | 前者为约定责任，后者为填补实际损失的赔偿 |
| 笼统说"担保" | 保证 / 抵押 / 质押 / 留置 / 定金 | "担保"是上位概念，须点明具体方式 |
| 解除与终止不分 | 解除 / 终止 | 解除可溯及既往；终止仅向将来消灭权利义务 |
| 意外事件当免责事由 | 不可抗力 | 不可抗力须"不能预见、不能避免、不能克服"，意外事件非法定免责事由 |
| 撤销权说"诉讼时效" | 除斥期间 | 形成权适用除斥期间（不变期间）；请求权才适用诉讼时效 |
| 善意=善良 | 善意 | 法律上指"不知道且不应当知道"，非道德评价 |
| 无效/可撤销/效力待定混称 | 分别使用 | 三者法律后果迥异，不可互换 |
| 连带/按份/补充责任混用 | 分别使用 | 责任形态直接决定当事人负担范围 |

### 刑事

| ❌ 易错 | ✅ 规范 | 辨析要点 |
|---|---|---|
| 审判阶段仍称"犯罪嫌疑人" | 侦查、审查起诉阶段称犯罪嫌疑人；审判阶段称被告人 | 阶段不同、称谓不同 |
| 被动归案说成"自首" | 自首 / 坦白 | 自首须自动投案＋如实供述；被动归案后如实供述为坦白 |
| 罚金当行政"罚款" | 罚金（刑事附加刑）/ 罚款（行政处罚） | 性质根本不同，不可跨部门混用 |
| 缓刑写成"缓期执行" | 缓刑 / 死刑缓期执行 | "缓期执行"特指死缓，与暂缓执行刑罚的缓刑不是一回事 |
| 故意/过失笼统带过 | 直接故意 / 间接故意 / 疏忽大意的过失 / 过于自信的过失 | 主观方面四种形态须分别表述 |
| 主犯/从犯/胁从犯不分 | 分别使用 | 共同犯罪角色影响量刑 |
| 追诉时效与诉讼时效混用 | 追诉时效（刑事）/ 诉讼时效（民事） | 分属不同部门 |

### 行政

| ❌ 易错 | ✅ 规范 | 辨析要点 |
|---|---|---|
| 处罚与处分混用 | 行政处罚（对相对人）/ 行政处分（对公职人员） | 对象不同 |
| "行政审批" | 行政许可 | 规范文本应用法定的"行政许可" |
| 行政复议与行政诉讼混称 | 分别使用 | 救济途径不同 |
| "具体行政行为" | 行政行为 | 现行《行政诉讼法》已统一表述，不再区分具体/抽象 |
| 吊销/撤销/注销混用 | 分别使用 | 吊销为处罚、撤销为纠错、注销为程序终结 |

### 诉讼与程序

| ❌ 易错 | ✅ 规范 | 辨析要点 |
|---|---|---|
| 原告与申请人混用 | 原告（诉讼）/ 申请人（仲裁、非讼） | 程序不同、称谓不同 |
| 上诉与申请再审不分 | 分别使用 | 上诉针对未生效裁判；申请再审针对已生效裁判 |
| 裁定/判决/决定混用 | 分别使用 | 裁定管程序、判决管实体、决定管诉讼指挥 |
| 驳回起诉与驳回诉讼请求混称 | 分别使用 | 前者为不符合起诉条件（裁定）；后者为实体无理由（判决） |
| 中止/终结/中断不分 | 分别使用 | 中止为暂停、终结为结束、中断为时效概念 |

### 合同与商事

| ❌ 易错 | ✅ 规范 | 辨析要点 |
|---|---|---|
| 通篇"甲方/乙方" | 出租人/承租人、出卖人/买受人、委托人/受托人等 | 宜用法定称谓，首次出现时括注简称 |
| 要约与要约邀请不分 | 分别使用 | 要约有约束力，要约邀请没有 |
| 不可抗力与情势变更混用 | 分别使用 | 前者导致不能履行；后者导致继续履行显失公平 |
| 代位权与撤销权混用 | 分别使用 | 代位权向次债务人行使；撤销权针对债务人不当处分 |
| 股民/投资者 | 股东 | 法律文本用"股东" |
| "法人代表" | 法定代表人 | "法人代表"不是法律术语 |

### 通用与立法技术

| ❌ 易错 | ✅ 规范 | 辨析要点 |
|---|---|---|
| 权利与权力不分 | 权利（私法主体）/ 权力（公权力机关） | 概念分属私法与公法 |
| 把法人等同于其负责人 | 法人 / 法定代表人 / 自然人 | 法人是组织体，不等于其负责人 |
| 赔偿与补偿混用 | 赔偿（针对违法行为）/ 补偿（针对合法行为造成的损失） | 用错会暗示行为定性错误 |
| 法律/法规/规章不分效力 | 按效力层级分别使用 | 立法主体不同、位阶不同 |
| 应当/可以/必须随意用 | 按规范性质分别使用 | "应当"为义务、"可以"为授权、"必须"语气最强 |
| "以上""以下"不交代本数 | 明确是否含本数 | 刑法、民法中通常包含本数，仍宜点明 |
| "日"不分自然日/工作日 | 明确指定 | 期间计算口径不同 |
| 知识产权笼统使用 | 著作权/专利权/商标权等按具体类型 | "知识产权"是上位概念 |

## 五、工作流程

按以下步骤处理一段文本：

1. **通读标记**：逐句扫描，圈出所有法律术语及疑似术语，重点盯住主体称谓、法律行为（解除/终止/撤销/撤回）、责任形态、权利类型、程序用语、数量与期限表达。
2. **定位语境**：判断文本所属法律部门（民事/刑事/行政/商事等）与文书类型，因为同一词在不同部门含义可能不同；裁判文书要求最严，合同允许约定简称，备忘录相对灵活但核心术语仍须规范。
3. **逐词核验**：对每个术语依"择词依据顺位"核对——是否有法定表述、在本部门语境中含义是否正确、外延是否精确、是否为现行法所用、是否落入第四节对照表中的混淆项。
4. **一致性比对**：建立全文术语索引，确认同一概念用词统一；使用简称的，须在首次出现处定义（如"出卖人甲公司，以下简称甲方"），避免前后换词制造歧义。
5. **文体打磨**：清除口语、俗称、网络用语、修辞和情绪化措辞，把模糊限定词替换为精确表达（见下表）。
6. **时效校验**：确认用语与现行有效法律一致，剔除已废止法律中的旧词。
7. **产出**：给出修正后的完整文本，并附逐条修正说明。

### 文体替换示例

| 应避免 | 规范替代 |
|---|---|
| 打官司 | 提起诉讼 |
| 大概、左右、若干 | 具体数额、具体日期 |
| 天价赔偿 | 赔偿金额为人民币×元 |
| 恶劣行径 | 违法行为 |
| 老赖 | 失信被执行人 |
| 不用负责 | 不承担相应责任 |

### 应留意的法律变迁（旧词→新词）

经济合同→合同；具体行政行为→行政行为；二年普通诉讼时效→三年；民事行为能力的笼统表述→完全/限制/无民事行为能力。处理旧文本时尤须注意。

## 六、置信度与产出格式

对每一处修正标注置信度，便于复核：**确定**（有明确法条支持，可直接采用）、**高**（有司法解释或通说支持）、**中**（基于实务惯例的择优，建议确认）、**低**（存在争议或地方差异，标为建议并说明理由）。

修正说明建议采用下表结构：

| 序号 | 位置 | 原表达 | 规范表达 | 类型 | 理由 | 等级 | 置信度 |
|---|---|---|---|---|---|---|---|
| 1 | 第×条 | … | … | 术语错误/不精确/混淆/过时/文体不当/前后不一 | … | 高危/提示/建议 | 确定 |

等级含义：**高危**——不改可能影响法律效力或造成重大误解，必须修正；**提示**——可能产生歧义或欠专业，建议修正；**建议**——文体优化，可选。

## 七、应避免的典型陷阱

- **望文生义**：按日常含义理解术语（如把"善意取得"的"善意"理解成道德善良）。以法律定义为准。
- **近义替换**：为避免重复而换词（把"解除"改写成"取消""废除"）。同一概念坚持同一术语。
- **跨部门移植**：把一个部门的词用到另一部门（民事文本里写当事人"受处罚"）。确认术语与语境同属一个部门。
- **旧法惯性**：续用已被新法替换的旧词。关注修订动态。
- **上位概念替代**：该精确时用上位词笼统带过（用"担保"代替本应明确的"抵押""质押"）。需要精确时取最具体的下位概念。
- **边界含糊**："以上""以下""日"等不交代口径。按法律规定标明是否含本数、是否为工作日。

## 八、快速核对口诀

一查法条：有无法定表述？二查部门：是否属于正确的法律部门？三查语境：含义与上下文是否吻合？四查一致：同一概念全文是否统一？五查时效：是否与现行法一致？六查文体：是否符合文书规范？七查精度：有无模糊、笼统、歧义？

绝不可犯的红线：犯罪嫌疑人/被告人随阶段混用、定金/订金混淆、缓刑/死缓混淆、自首/坦白不分、"法人代表"（应为法定代表人）、罚金/罚款跨部门混用、赔偿/补偿混淆、诉讼时效/除斥期间混淆。
